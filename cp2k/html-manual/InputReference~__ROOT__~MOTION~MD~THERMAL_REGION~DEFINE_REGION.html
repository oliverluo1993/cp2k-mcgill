Section DEFINE_REGION

Index of all sections. This section is located at __ROOT__%MOTION%MD%THERMAL_REGION%DEFINE_REGION.

This section provides the possibility to define arbitrary region. This section can be repeated.

Subsections

None

Section keywords

  • LIST
  • TEMPERATURE
  • TEMP_TOL
  • DO_LANGEVIN

Keyword descriptions

LIST
LIST {integer} {integer} .. {integer}
Specifies a list of atoms belonging to the region.
This keyword expects a list of integers, and may repeat
TEMPERATURE
temperature 5.0
The temperature in K used to initialize the velocities of the atoms in this region
This keyword expects a real. Default unit: [K], default value: 0.0000000000000000E+00 K
TEMP_TOL
temp_tol 0.0
Maximum accepted temperature deviation from the expected value for this region. If temp_tol=0 no rescaling is performed
This keyword expects a real. Default unit: [K], default value: 0.0000000000000000E+00 K
DO_LANGEVIN
do_langevin .TRUE.
When ENSEMBLE is set to LANGEVIN, Controls whether the atoms in the thermal region should undergo Langevin MD. If not, then they will undergo NVE Born-Oppenheimer MD.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
NOISY_GAMMA_REGION
noisy_gamma_region 4.0E-5
Special imaginary Langevin Friction term for Langevin Dynamics with noisy forces for the atoms in this region.When set, overrides the general value set by NOISY_GAMMA in the MOTION%MD%LANGEVIN section.When unset for a defined region, the general NOISY_GAMMA value applies.
This keyword expects a real. Default unit: [fs^-1].
This keyword cites following reference: [Kuhne2007]
