Section ALMO_OPTIMIZER_PCG

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%ALMO_SCF%ALMO_OPTIMIZER_PCG.

Controls the PCG optimization of block-diagonal ALMOs. This section can not be repeated.

Maximum number of iterations

EPS_ERROR
EPS_ERROR 1.E-6
Target value of the MAX norm of the error
This keyword expects a real. Default value: 1.0000000000000001E-05
MAX_ITER_EARLY
MAX_ITER_EARLY 5
Maximum number of iterations for truncated SCF (e.g. Langevin-corrected MD). Negative values mean that this keyword is not used.
This keyword expects an integer. Default value: -1
EPS_ERROR_EARLY
EPS_ERROR_EARLY 1.E-2
Target value of the MAX norm of the error for truncated SCF (e.g. Langevin-corrected MD). Negative values mean that this keyword is not used.
This keyword expects a real. Default value: -1.0000000000000000E+00
LIN_SEARCH_EPS_ERROR
LIN_SEARCH_EPS_ERROR 1.E-2
Target value of the gradient norm during the linear search
This keyword expects a real. Default value: 1.0000000000000000E-03
LIN_SEARCH_STEP_SIZE_GUESS
LIN_SEARCH_STEP_SIZE_GUESS 0.1
The size of the first step in the linear search
This keyword expects a real. Default value: 1.0000000000000000E+00
CONJUGATOR
CONJUGATOR POLAK_RIBIERE
Various methods to compute step directions in the PCG optimization
This keyword expects a keyword. Default value: HAGER_ZHANG
valid keywords:
  • ZERO: Steepest descent
  • POLAK_RIBIERE: Polak and Ribiere
  • FLETCHER_REEVES: Fletcher and Reeves
  • HESTENES_STIEFEL: Hestenes and Stiefel
  • FLETCHER: Fletcher (Conjugate descent)
  • LIU_STOREY: Liu and Storey
  • DAI_YUAN: Dai and Yuan
  • HAGER_ZHANG: Hager and Zhang
MAX_ITER_OUTER_LOOP
MAX_ITER 10
Maximum number of iterations in the outer loop. Use the outer loop to update the preconditioner and reset the conjugator. This can speed up convergence significantly.
This keyword expects an integer. Default value: 0
PRECONDITIONER
PRECONDITIONER DOMAIN
Select a preconditioner for the conjugate gradient optimization
This keyword expects a keyword. Default value: DEFAULT
valid keywords:
  • NONE: Do not use preconditioner
  • DEFAULT: Same as DOMAIN preconditioner
  • DOMAIN: Invert preconditioner domain-by-domain. The main component of the linear scaling algorithm
  • FULL: Solve linear equations step=-H.grad on the entire space
