Keyword descriptions

K
K {real}
Specifies the force constant for the harmonic restraint. The functional form for the restraint is: K*(X-TARGET)^2.
This keyword expects a real. Default unit: [internal_cp2k], default value: 0.0000000000000000E+00 internal_cp2k
