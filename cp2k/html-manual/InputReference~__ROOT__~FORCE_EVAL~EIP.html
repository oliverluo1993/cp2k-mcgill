Keyword descriptions

EIP_MODEL
EIP_MODEL BAZANT
Selects the empirical interaction potential model
This keyword expects a keyword. Default value: LENOSKY
valid keywords:
  • BAZANT: Bazant potentials
  • EDIP: Environment-Dependent Interatomic Potential
  • LENOSKY: Lenosky potentials
variants: EIP-MODEL
