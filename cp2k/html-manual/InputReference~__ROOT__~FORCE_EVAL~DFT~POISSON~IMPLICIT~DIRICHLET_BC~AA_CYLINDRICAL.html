Section AA_CYLINDRICAL

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%POISSON%IMPLICIT%DIRICHLET_BC%AA_CYLINDRICAL.

Parameters for creating axis-aligned cylindrical Dirichlet boundary regions. This section can be repeated.

Keyword descriptions

V_D
v_D
The value of the fixed potential to be imposed at the cylindrical Dirichlet boundary.
This keyword expects a real. Default unit: [volt].
OSCILLATING_FRACTION
OSCILLATING_FRACTION
A fraction of the field can be set to oscilate over time.
This keyword expects a real. Default value: 0.0000000000000000E+00
FREQUENCY
FREQUENCY
The frequency with which the oscillating fraction oscillates.
This keyword expects a real. Default unit: [s^-1], default value: 0.0000000000000000E+00 s^-1
PHASE
PHASE
The phase of the oscillattion. A phase of zero corresponds to a cosine function.
This keyword expects a real. Default value: 0.0000000000000000E+00
PARALLEL_AXIS
PARALLEL_AXIS
The coordinate axis that the cylindrical region extends along.
This keyword expects a keyword
valid keywords: X, Y, Z
XTNT
xtnt
The extents of the cylinder along its central axis.
This keyword expects precisely 2 reals. Default unit: [angstrom].
BASE_CENTER
BASE_CENTER
The y and z coordinates of the cylinder's base center.
This keyword expects precisely 2 reals. Default unit: [angstrom].
BASE_RADIUS
BASE_RADIUS
The base radius of the cylinder.
This keyword expects a real. Default unit: [angstrom], default value: 5.2917720859000006E-01 angstrom
N_SIDES
N_SIDES
The number of sides (faces) of the n-gonal prism approximating the cylinder.
This keyword expects an integer. Default value: 5
APX_TYPE
APX_TYPE
Specifies the type of the n-gonal prism approximating the cylinder.
This keyword expects a keyword. Default value: CIRCUMSCRIBED
valid keywords: CIRCUMSCRIBED, INSCRIBED
N_PRTN
N_PRTN
The number of partitions along the face edges of the prism for tiling. If the edges have different lengths, from the two given values, the larger one will be assigned to the longer edge.
This keyword expects precisely 2 integers. Default value: 1 1
THICKNESS
THICKNESS
The thickness of the cylinder.
This keyword expects a real. Default unit: [angstrom], default value: 3.9688290644250002E-01 angstrom
SMOOTHING_WIDTH
SMOOTHING_WIDTH
The width of the transition region between the Dirichlet subdomain and its surrounding.
This keyword expects a real. Default unit: [angstrom], default value: 1.0583544171800002E-01 angstrom
variants: SIGMA
DELTA_ALPHA
delta_alpha
A central angle specifying the gap between the faces of the n-gonal prism. To avoide overlap between the cuboids (of the given thickness) built on top of the faces, a larger value is required if the number of faces (N_SIDES) is quite few and/or the base radius is fairly small.
This keyword expects a real. Default unit: [rad], default value: 5.0000000000000003E-02 rad
