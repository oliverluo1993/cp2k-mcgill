Section OPBEND

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%MM%FORCEFIELD%OPBEND.

Specifies the out of plane bend potential of the MM system.(Only defined for atom quadruples which are also defined as an improper pattern in the topology.). This section can be repeated.

Subsections

None

Section keywords

  • ATOMS
  • KIND
  • K
  • PHI0

Keyword descriptions

ATOMS
ATOMS {KIND1} {KIND2} {KIND3} {KIND4}
Defines the atomic kinds involved in the opbend.
This keyword expects precisely 4 words
KIND
KIND HARMONIC
Define the kind of out of plane bend potential
This keyword expects a keyword. Default value: HARMONIC
valid keywords:
  • HARMONIC: Functional Form (HARMONIC): 0.5 * K * [ PHI - PHI0 ]**2
  • MM2: Functional Form (MM2|MM3|MM4): K * [ PHI - PHI0 ]**2
  • MM3: Functional Form (MM2|MM3|MM4): K * [ PHI - PHI0 ]**2
  • MM4: Functional Form (MM2|MM3|MM4): K * [ PHI - PHI0 ]**2
K
K {real}
Defines the force constant of the potential
This keyword expects a real. Default unit: [rad^-2*hartree].
PHI0
PHI0 {real}
Defines the phase of the potential.
This keyword expects a real. Default unit: [rad].
