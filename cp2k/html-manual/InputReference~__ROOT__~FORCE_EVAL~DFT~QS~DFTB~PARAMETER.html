Section PARAMETER

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%QS%DFTB%PARAMETER.

Information on where to find DFTB parameters. This section can not be repeated.

Keyword descriptions

SK_FILE
SK_FILE a1 a2 filename
Define parameter file for atom pair
This keyword expects precisely 3 words, and may repeat
PARAM_FILE_PATH
PARAM_FILE_PATH pathname
Specify the directory with the DFTB parameter files. Used in combination with the filenames specified in the file given in PARAM_FILE_NAME.
This keyword expects a word. Default value: ./
PARAM_FILE_NAME
PARAM_FILE_NAME filename
Specify file that contains the names of Slater-Koster tables: A plain text file, each line has the format "ATOM1 ATOM2 filename.spl".
This keyword expects a word. Default value: " "
DISPERSION_TYPE
DISPERSION_TYPE (UFF|D3)
Use dispersion correction of the specified type. Dispersion correction has to be switched on in the DFTB section.
This keyword expects a keyword. Default value: UFF
valid keywords:
  • UFF: Uses the UFF force field for a pair potential dispersion correction.
  • D3: Uses the Grimme D3 method (simplified) for a pair potential dispersion correction.
UFF_FORCE_FIELD
UFF_FORCE_FIELD filename
Name of file with UFF parameters that will be used for the dispersion correction. Needs to be specified when DISPERSION==.TRUE., otherwise cp2k crashes with a Segmentation Fault.
This keyword expects a word. Default value: " "
DISPERSION_PARAMETER_FILE
DISPERSION_PARAMETER_FILE filename
Specify file that contains the atomic dispersion parameters for the D3 method
This keyword expects a word. Default value: " "
DISPERSION_RADIUS
DISPERSION_RADIUS
Define radius of dispersion interaction
This keyword expects a real. Default value: 1.5000000000000000E+01
COORDINATION_CUTOFF
COORDINATION_CUTOFF
Define cutoff for coordination number calculation
This keyword expects a real. Default value: 9.9999999999999995E-07
D3_SCALING
D3_SCALING 1.0 1.0 1.0
Scaling parameters (s6,sr6,s8) for the D3 dispersion method,
This keyword expects precisely 3 reals. Default value: 0.0000000000000000E+00 0.0000000000000000E+00 0.0000000000000000E+00
HB_SR_PARAM
HB_SR_PARAM {real}
Uses a modified version for the GAMMA within the SCC-DFTB scheme, specifically tuned for hydrogen bonds. Specify the exponent used in the exponential.
This keyword expects a real. Default value: 4.0000000000000000E+00
