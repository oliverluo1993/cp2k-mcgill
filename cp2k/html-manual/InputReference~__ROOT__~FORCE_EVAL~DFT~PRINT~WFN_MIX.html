Section WFN_MIX

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%PRINT%WFN_MIX.

A section that allows manipulation of the MO coeffs, e.g. for changing a ground state into an excited state.Starting from a copy of the original MOs, changes can be madeby adding linear combinations of HOMO/LUMO of the original MOs to the result MOs. This section can not be repeated.

Keyword descriptions

OVERWRITE_MOS
If the keyword is active molecular orbitals in memory will be replaced by the mixed wfn. In combination with RTP or EMD no restart will be required to use the mixed wfn.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
