Section MOVE_TYPE

Index of all sections. This section is located at __ROOT__%MOTION%TMC%NMC_MOVES%MOVE_TYPE.

This section specifies the TMC move type, and its properties. Selectable types are: ATOM_TRANS atom translation, MOL_TRANS molecule translation, MOL_ROT molecule rotation, PROT_REORDER proton reordering, PT_SWAP Parallel Tempering swap, VOL_MOVE volume change, ATOM_SWAP swaps two atoms of different type. This section can be repeated.

Keyword descriptions

_SECTION_PARAMETERS_
ATOM_TRANS
The name of the move type described in this section.
This keyword expects a word. Default value: DEFAULT
SIZE
SIZE {real}
Defines the size of the move:ATOM_TRANS [A], MOL_TRANS [A], MOL_ROT [degree], PROT_REORDER [], VOL_MOVE [A], ATOM_SWAP
This keyword expects a real. Default value: -1.0000000000000000E+00
PROB
PROB {real}
Defines the probability of the move (considering the ration between the selected moves)
This keyword expects a real. Default value: 1.0000000000000000E+00
INIT_ACC_PROB
INIT_ACC_PROB {real}
Defines the initial probability of accepting the move.
This keyword expects a real. Default value: 2.3000000000000001E-01
ATOMS
ATOMS {KIND1} {KIND2} ...
Defines the atomic kinds involved in the move. Up to now only used for the atom swap.
This keyword expects a list of words, and may repeat
