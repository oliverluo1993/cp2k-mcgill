Keyword descriptions

METHOD
METHOD MP2_GPW
Which method should be used to compute the MP2 energy
This keyword expects a keyword. Default value: DIRECT_CANONICAL
valid keywords:
  • NONE: Skip MP2 calculation
  • DIRECT_CANONICAL: Use the direct mp2 canonical approach
  • MP2_GPW: Use the GPW approach to MP2
  • RI_MP2_GPW: Use the GPW approach to RI-MP2
  • RI_RPA_GPW: Use the GPW approach to RI-RPA
  • RI_SOS_LAPLACE: Use the GPW approach to RI-SOS-Laplace-MP2
  • OPTIMIZE_RI_BASIS: Optimize RIMP2 basis set

This keyword cites following references: [DelBen2012] [DelBen2013]
MEMORY
MEMORY 1500
Maximum allowed total memory usage during MP2 methods [MiB].
This keyword expects a real. Default value: 1.0240000000000000E+03
SCALE_S
SCALE_S 1.0
Scaling factor of the singlet energy component (opposite spin, OS).
This keyword expects a real. Default value: 1.0000000000000000E+00
SCALE_T
SCALE_T 1.0
Scaling factor of the triplet energy component (same spin, SS).
This keyword expects a real. Default value: 1.0000000000000000E+00
GROUP_SIZE
GROUP_SIZE 32
Group size used in the computation of the integrals. Default is to use all processors (GROUP_SIZE=-1).A smaller group size (for example the node size), might a better choice if the actual MP2 time is large compared to integral computation time. This is usually the case if the total number of processors is not too large.
This keyword expects an integer. Default value: -1
variants: NUMBER_PROC
ROW_BLOCK
ROW_BLOCK 512
Size of the row block used in the SCALAPACK block cyclic data distribution.Default is (ROW_BLOCK=-1) is automatic. A proper choice can speedup the parallel matrix multiplication in the case of RI-RPA and RI-SOS-MP2-Laplace.
This keyword expects an integer. Default value: -1
variants: ROW_BLOCK_SIZE
COL_BLOCK
COL_BLOCK 512
Size of the column block used in the SCALAPACK block cyclic data distribution.Default is (COL_BLOCK=-1) is automatic. A proper choice can speedup the parallel matrix multiplication in the case of RI-RPA and RI-SOS-MP2-Laplace.
This keyword expects an integer. Default value: -1
variants: COL_BLOCK_SIZE
CALC_COND_NUM
CALC_COND_NUM
Calculate the condition number of the (P|Q) matrix for the RI methods.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
variants: CALC_CONDITION_NUMBER
RI_METRIC
RI_METRIC COULOMB
Define which RI metric should be used for calculating B^P_ia.
This keyword expects a keyword. Default value: COULOMB
valid keywords:
  • COULOMB: Use Coulomb metric.
  • OVERLAP: Use overlap metric, recommended in combination with imaginary time for O(N^3) RPA.
variants: RI
ERI_METHOD
ERI_METHOD MME
Method for calculating periodic electron repulsion integrals (MME method is faster but experimental, forces and 3-center integrals not yet implemented).
This keyword expects a keyword. Default value: GPW
valid keywords:
  • GPW: Uses Gaussian Plane Wave method [DelBen2013]
  • MME: Uses MiniMax-Ewald method (ERI_MME subsection)
