Keyword descriptions

TYPE
TYPE DRSLL
Type of functional (the corresponding kernel data file should be selected).Allows for common forms such as vdW-DF, vdW-DF2, optB88-vdW, rVV10.
This keyword expects a keyword. Default value: DRSLL
valid keywords:
  • DRSLL: Dion-Rydberg-Schroeder-Langreth-Lundqvist nonlocal van der Waals density functional
  • LMKLL: Lee-Murray-Kong-Lundqvist-Langreth nonlocal van der Waals density functional
  • RVV10: Revised Vydrov-van Voorhis nonlocal van der Waals density functional

This keyword cites following reference: [Tran2013]

KERNEL_FILE_NAME

This keyword expects a string. Default value: vdW_kernel_table.dat
