Keyword descriptions

NPROC_REP
Specify the number of processors to be used per replica environment (for parallel runs)
This keyword expects an integer. Default value: 1
PROC_DIST_TYPE
PROC_DIST_TYPE (INTERLEAVED|BLOCKED)
Specify the topology of the mapping of processors into replicas.
This keyword expects a keyword. Default value: BLOCKED
valid keywords:
  • INTERLEAVED: Interleaved distribution
  • BLOCKED: Blocked distribution
BAND_TYPE
BAND_TYPE (B-NEB|IT-NEB|CI-NEB|D-NEB|SM|EB)
Specifies the type of BAND calculation
This keyword expects a keyword. Default value: IT-NEB
valid keywords:
  • B-NEB: Bisection nudged elastic band
  • IT-NEB: Improved tangent nudged elastic band
  • CI-NEB: Climbing image nudged elastic band
  • D-NEB: Doubly nudged elastic band
  • SM: String Method
  • EB: Elastic band (Hamiltonian formulation)
NUMBER_OF_REPLICA
Specify the number of Replica to use in the BAND
This keyword expects an integer. Default value: 10
USE_COLVARS
Uses a version of the band scheme projected in a subspace of colvars.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
POT_TYPE
POT_TYPE (FULL|FE|ME)
Specifies the type of potential used in the BAND calculation
This keyword expects a keyword. Default value: FULL
valid keywords:
  • FULL: Full potential (no projections in a subspace of colvars)
  • FE: Free energy (requires a projections in a subspace of colvars)
  • ME: Minimum energy (requires a projections in a subspace of colvars)
ROTATE_FRAMES
Compute at each BAND step the RMSD and rotate the frames in order to minimize it.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
ALIGN_FRAMES
Enables the alignment of the frames at the beginning of a BAND calculation. This keyword does not affect the rotation of the replicas during a BAND calculation.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
K_SPRING
Specify the value of the spring constant
This keyword expects a real. Default value: 2.0000000000000000E-02
variants: K
