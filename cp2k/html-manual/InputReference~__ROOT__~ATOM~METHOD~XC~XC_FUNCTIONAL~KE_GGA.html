Section KE_GGA

Index of all sections. This section is located at __ROOT__%ATOM%METHOD%XC%XC_FUNCTIONAL%KE_GGA.

Uses one of the KE_GGA functionals (optimized versions of some of these functionals might be available outside this section). These functionals are needed for the computation of the kinetic energy in the Kim-Gordon method. This section can be repeated.

Subsections

None

Keyword descriptions

_SECTION_PARAMETERS_
activates the functional
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
FUNCTIONAL
FUNCTIONAL (OL1|OL2|LLP|PW86|PW91|LC|T92|PBE)
Which one of the KE_GGA functionals should be used
This keyword expects a keyword. Default value: LLP
valid keywords:
  • OL1: Uses first Ou-Yang and Levy functional, currently not producing correct results
  • OL2: Uses second Ou-Yang and Levy functional, currently not producing correct results
  • LLP: Uses Lee, Lee, and Parr functional
  • PW86: Uses Perdew and Wang's 1986 functional
  • PW91: Uses Perdew and Wang's 1991 functional
  • LC: Uses Lembarki and Chermette functional
  • T92: Uses Thakkar functional
  • PBE: Uses the 1996 functional of Perdew, Burke and Ernzerhof
