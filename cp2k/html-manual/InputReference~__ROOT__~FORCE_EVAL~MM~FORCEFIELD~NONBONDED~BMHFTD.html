Section BMHFTD

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%MM%FORCEFIELD%NONBONDED%BMHFTD.

This section specifies the input parameters for BMHFTD potential type.Functional form: V(r) = A*exp(-B*r) - f_6*(r)C/r^6 - f_8(r)*D/r^8.where f_order(r)=1-exp(-BD * r) * \sum_{k=0}^order (BD * r)^k / k! .(Tang-Toennies damping function)No values available inside cp2k. This section can be repeated.

This section cites following references: [FUMI1964] [TOSI1964]
