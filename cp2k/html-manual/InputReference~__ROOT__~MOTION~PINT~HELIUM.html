The CP2K project : input section HELIUM

Section HELIUM

Index of all sections. This section is located at __ROOT__%MOTION%PINT%HELIUM.

The section that controls optional helium solvent environment (highly experimental, not for general use yet). This section can not be repeated.

Keyword descriptions

_SECTION_PARAMETERS_
silent
Whether or not to actually use this section
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
HELIUM_ONLY
Simulate helium solvent only, disregard solute entirely
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
INTERACTION_POT_SCAN
Scan solute-helium interaction potential, cubefile parameters set in subsection RHO
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
NUM_ENV
Number of independent helium environments
This keyword expects an integer. Default value: 1
POTENTIAL_FILE_NAME
Name of the Helium interaction potential file
This keyword expects a string. Default value: HELIUM.POT
GET_FORCES
GET_FORCES (AVERAGE|LAST)
Get average MC forces or last MC forces to propagate MD
This keyword expects a keyword. Default value: AVERAGE
valid keywords: AVERAGE, LAST
SOLUTE_INTERACTION
SOLUTE_INTERACTION (NONE | MWATER)
Interaction potential between helium and the solute
This keyword expects a keyword. Default value: NONE
valid keywords:
  • NONE: No interaction with solute
  • MWATER: Test interaction with wrong Water
NATOMS
Number of helium atoms
This keyword expects an integer. Default value: 64
NBEADS
Number of helium path integral beads
This keyword expects an integer. Default value: 25
RNG_SEED
RNG_SEED
Initial seed for the (pseudo)random number generator that controls helium coordinate generation and propagation.
This keyword expects an integer. Default value: 12345
N_INNER
Number of MC iterations at the same time slice(s) (number of inner MC loop iterations)
This keyword expects an integer. Default value: 6600
variants: INOROT
N_OUTER
how often to reselect the time slice(s) to work on (number of outer MC loop iterations)
This keyword expects an integer. Default value: 300
variants: IROT
BISECTION
how many time slices to change at once (+1). Must be a power of 2 currently
This keyword expects an integer. Default value: 8
MAX_PERM_CYCLE
how large cyclic permutations to try
This keyword expects an integer. Default value: 6
SAMPLING_METHOD
SAMPLING_METHOD (CEPERLEY)
Choose between Ceperley or other not implemented algorithms
This keyword expects a keyword. Default value: CEPERLEY
valid keywords: CEPERLEY
PERIODIC
Use periodic boundary conditions for helium
This keyword expects a logical. Default value: F
CELL_SIZE
PBC unit cell size (NOTE 1: density, number of atoms and volume are interdependent - give only two of them; NOTE 2: for small cell sizes specify NATOMS instead)
This keyword expects a real. Default unit: [angstrom].
CELL_SHAPE
CELL_SHAPE (CUBE|OCTAHEDRON)
PBC unit cell shape for helium
This keyword expects a keyword. Default value: CUBE
valid keywords: CUBE, OCTAHEDRON
DROPLET_RADIUS
Reject a move if any of the new positions does not lie within this range from the center of gravity
This keyword expects a real. Default unit: [angstrom], default value: 9.5129823500784665+307 angstrom
DENSITY
trial density of helium for determining the helium box size
This keyword expects a real. Default unit: [angstrom^-3], default value: 2.1859999999999997E-02 angstrom^-3
PRESAMPLE
Presample He coordinates before first PIMD step
This keyword expects a logical. Default value: F
