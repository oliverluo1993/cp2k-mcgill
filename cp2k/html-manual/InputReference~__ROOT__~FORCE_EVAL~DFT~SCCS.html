Section SCCS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%SCCS.

Define the parameters for self-consistent continuum solvation (SCCS) model. This section can not be repeated.

Keyword descriptions

_SECTION_PARAMETERS_
&SCCS ON
Controls the activation of the SCCS section
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
ALPHA
Solvent specific tunable parameter for the calculation of the repulsion term Grep = α S where S is the (quantum) surface of the cavity
This keyword expects a real. Default unit: [mN*m^-1], default value: 0.0000000000000000E+00 mN*m^-1
BETA
Solvent specific tunable parameter for the calculation of the dispersion term Gdis = β V where V is the (quantum) volume of the cavity
This keyword expects a real. Default unit: [GPa], default value: 0.0000000000000000E+00 GPa
DELTA_RHO
Numerical increment for the calculation of the (quantum) surface of the solute cavity
This keyword expects a real. Default value: 2.0000000000000002E-05
DERIVATIVE_METHOD
DERIVATIVE_METHOD cd5
Method for the calculation of the numerical derivatives on the real-space grids
This keyword expects a keyword. Default value: FFT
valid keywords:
  • FFT: Fast Fourier transformation
  • CD3: 3-point stencil central differences
  • CD5: 5-point stencil central differences
  • CD7: 7-point stencil central differences
DIELECTRIC_CONSTANT
DIELECTRIC_CONSTANT 78.36
Dielectric constant of the solvent
This keyword expects a real. Default value: 8.0000000000000000E+01
variants: EPSILON_SOLVENT
EPS_SCCS
EPS_ITER 1.0E-7
Tolerance for the convergence of the polarisation density, i.e. requested accuracy for the SCCS iteration cycle
This keyword expects a real. Default value: 9.9999999999999995E-07
variants: EPS_ITER, TAU_POL
EPS_SCF
EPS_SCF 1.0E-2
The SCCS iteration cycle is activated only if the SCF iteration cycle is converged to this threshold value
This keyword expects a real. Default value: 5.0000000000000000E-01
GAMMA
Surface tension of the solvent used for the calculation of the cavitation term Gcav = γ S where S is the (quantum) surface of the cavity
This keyword expects a real. Default unit: [mN*m^-1], default value: 0.0000000000000000E+00 mN*m^-1
variants: SURFACE_TENSION
MAX_ITER
MAX_ITER 50
Maximum number of SCCS iteration steps performed to converge within the given tolerance
This keyword expects an integer. Default value: 100
METHOD
METHOD Fattebert-Gygi
Method used for the smoothing of the dielectric function
This keyword expects a keyword. Default value: ANDREUSSI
valid keywords:
  • ANDREUSSI: Smoothing function proposed by Andreussi et al.
  • FATTEBERT-GYGI: Smoothing function proposed by Fattebert and Gygi
MIXING
MIXING 0.2
Mixing parameter (Hartree damping) employed during the iteration procedure
This keyword expects a real. Default value: 5.9999999999999998E-01
variants: ETA
