Section EFIELD

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%EFIELD.

parameters for finite, time dependent, nonperiodic electric fields. This section can be repeated.

Keyword descriptions

INTENSITY
INTENSITY 0.001
Intensity of the electric field. For real-time propagation (RTP) units are in W*cm-2 which corresponds to a maximal amplitude in a.u. of sqrt(I/(3.50944*10^16)). For a constant local field in isolated system calclulations, units are in a.u..
This keyword expects a real. Default value: 0.0000000000000000E+00
POLARISATION
POLARISATION 0.0 0.0 1.0
Polarisation vector of electric field
This keyword expects precisely 3 reals. Default value: 0.0000000000000000E+00 0.0000000000000000E+00 1.0000000000000000E+00
WAVELENGTH
Wavelength 1.E0
Wavelength of efield field for real-time propagation (RTP) calculations.
This keyword expects a real. Default unit: [nm], default value: 0.0000000000000000E+00 nm
PHASE
Phase 1.E0
Phase offset of the cosine given in multiples of pi. Used in real-time propagation (RTP) calculations.
This keyword expects a real. Default value: 0.0000000000000000E+00
ENVELOP
ENVELOP CONSTANT
Shape of the efield pulse used in real-time propagation (RTP) calculations.
This keyword expects a keyword. Default value: CONSTANT
valid keywords:
  • CONSTANT: No envelop function is applied to the strength
  • GAUSSIAN: A Gaussian function is used as envelop
  • RAMP: Linear tune in/out of the field
  • CUSTOM: A custom field read from a file
