Section UPDATE

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%PRINT%WFN_MIX%UPDATE.

update a result MO with with a linear combination of of original MOs. This section can be repeated to build arbitrary linear combinations using repeatedly y=a*y+b*x. This section can be repeated.

None

Keyword descriptions

RESULT_MO_INDEX
RESULT_MO_INDEX 1
Index of the MO (y) to be modified. Counting down in energy with HOMO=1
This keyword expects an integer. Default value: 0
RESULT_MARKED_STATE
ORIG_MARKED_STATE 1
Specifies the MO according to the marks set in MOLECULAR_STATES. The value corresponds to the repetition of MARK_STATES in MOLECULAR_STATES
This keyword expects an integer. Default value: 0
RESULT_SPIN_INDEX
Spin of the MO (y) to be modified.
This keyword expects a keyword. Default value: ALPHA
valid keywords:
  • ALPHA: Majority spin
  • BETA: Minority spin
RESULT_SCALE
RESULT_SCALE 0.0
Scaling factor of the result variable (a).
This keyword expects a real
ORIG_MO_INDEX
ORIG_MO_INDEX 1
Index of the original MO (x). Counting down in energy with HOMO=1 or up from LUMO=1, depending on ORIG_IS_VIRTUAL.
This keyword expects an integer. Default value: 0
ORIG_MARKED_STATE
ORIG_MARKED_STATE 1
Specifies the MO according to the marks set in MOLECULAR_STATES. The value corresponds to the repetition of MARK_STATES in MOLECULAR_STATES
This keyword expects an integer. Default value: 0
ORIG_SPIN_INDEX
Spin of the MO (x) to be modified.
This keyword expects a keyword. Default value: ALPHA
valid keywords:
  • ALPHA: Majority spin
  • BETA: Minority spin
ORIG_SCALE
ORIG_SCALE 0.0
Scaling factor of the result variable (b).
This keyword expects a real
ORIG_IS_VIRTUAL
ORIG_IS_VIRTUAL
