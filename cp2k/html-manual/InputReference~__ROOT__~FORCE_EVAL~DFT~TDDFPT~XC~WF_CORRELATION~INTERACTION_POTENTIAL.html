Keyword descriptions

POTENTIAL_TYPE
POTENTIAL_TYPE TSHPSC
Which interaction potential should be used (Coulomb, TShPSC operator).
This keyword expects a keyword. Default value: COULOMB
valid keywords:
  • COULOMB: Coulomb potential: 1/r
  • TSHPSC: TShPSC:
    • 1/x - s/Rc for x ≤ Rc
    • (1 - s)/Rc - (x - Rc)/Rc^2 + (x - Rc)^2/Rc^3 - (2*n^2 - 7*n + 9 - 4*s)*(x - Rc)^3/(Rc^4*(n^2 - 2*n + 1)*(n - 1)) + (6-3*s - 4*n + n^2)*(x - Rc)^4/(Rc^5*(n^4 - 4*n^3 + 6*n^2 - 4*n + 1)) for Rc < x ≤ n*Rc (4th order polynomial)
    • 0 for x > n*Rc
TRUNCATION_RADIUS
TRUNCATION_RADIUS 10.0
Determines truncation radius for the truncated TShPSC potential. Only valid when doing truncated calculation
This keyword expects a real. Default unit: [angstrom], default value: 5.2917720859000008E+00 angstrom
TSHPSC_DATA
TShPSC_DATA t_sh_p_s_c.dat
Location of the file TShPSC.dat that contains the data for the evaluation of the TShPSC G0
This keyword expects a word. Default value: t_sh_p_s_c.dat
