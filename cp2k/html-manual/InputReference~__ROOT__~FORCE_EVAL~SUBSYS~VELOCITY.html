Section VELOCITY

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%VELOCITY.

The velocities for simple systems or the centroid mode in PI runs, xyz format by default. This section can not be repeated.

Subsections

None

Section keywords

  • _DEFAULT_KEYWORD_
  • PINT_UNIT

Keyword descriptions

_DEFAULT_KEYWORD_
{Real} {Real} {Real}
The atomic velocities in the format:

vx vy vz

The same order as for the atomic coordinates is assumed.
This keyword expects precisely 3 reals, and may repeat
PINT_UNIT
UNIT angstrom*au_t^-1
Specify the units of measurement for the velocities (currently works only for the path integral code). All available CP2K units can be used.
This keyword expects a word. Default value: bohr*au_t^-1
