Section SHG_INTEGRALS_TEST

Index of all sections. This section is located at __ROOT__%TEST%SHG_INTEGRALS_TEST.

Parameters for testing the SHG 2-center integrals for different r12 operators. Test w.r.t. performance and accurarcy. This section can not be repeated.
