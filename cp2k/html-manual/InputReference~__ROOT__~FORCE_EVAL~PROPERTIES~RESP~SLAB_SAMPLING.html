Section SLAB_SAMPLING

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%PROPERTIES%RESP%SLAB_SAMPLING.

Specifies the parameter for sampling the RESP fitting points for slab-like periodic systems, i.e. systems that involve surfaces. This section can only be used with periodic Poisson solver and cell. To see, which grid points were used, switch on COORD_FIT_POINTS in the PRINT section. This section can be repeated.

Subsections

None

Section keywords

  • ATOM_LIST
  • RANGE
  • LENGTH
  • SURF_DIRECTION

Keyword descriptions

ATOM_LIST
ATOM_LIST 1 2 3 or 1..3
Specifies the list of indexes of atoms used to define the region for the RESP fitting. The list should contain indexes of atoms of the first surface layer.
This keyword expects a list of integers, and may repeat
RANGE
RANGE
Range where the fitting points are sampled. A range of 3 to 5 Angstroms means that the fitting points are sampled in the region of 3 to 5 Angstroms above the surface which is defined by atom indexes given in ATOM_LIST.
This keyword expects precisely 2 reals. Default unit: [angstrom].
LENGTH
LENGTH
Length of the sampling box, i.e. a box of this length and the height specified by RANGE is defined above each surface atom given in ATOM_LIST. The grid points in the boxes are accepted as fitting point. Should be in the range of the nearest neighbour distance (a bit larger to be on the safe side). Allows for a refined sampling of grid points in case of corrugated surfaces.
This keyword expects a real. Default unit: [angstrom], default value: 3.0000000000000000E+00 angstrom
SURF_DIRECTION
SURF_DIRECTION Z
Specifies what above the surface means. Defines the direction.
This keyword expects a keyword. Default value: Z
valid keywords:
  • X: surface layers are piled up in x-direction
  • Y: surface layers are piled up in y-direction
  • Z: surface layers are piled up in z-direction
  • -X: surface layers are piled up in -x-direction
  • -Y: surface layers are piled up in -y-direction
  • -Z: surface layers are piled up in -z-direction
