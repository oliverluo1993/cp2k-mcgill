Section FIT_KIND

Index of all sections. This section is located at __ROOT__%OPTIMIZE_BASIS%FIT_KIND.

specicifies the atomic kinds to be fitted and the basis sets associated with the kind. This section can be repeated.

Keyword descriptions

_SECTION_PARAMETERS_
H
The name of the kind described in this section.
This keyword expects a word. Default value: DEFAULT
BASIS_SET
H
The name of the basis set for the kind. Has to be specified in BASIS_TEMPLATE_FILE.
This keyword expects a word. Default value: DEFAULT
INITIAL_DEGREES_OF_FREEDOM
INITIAL_DEGREES_OF_FREEDOM ALL
Specifies the initial degrees of freedom in the basis optimization.This can be used to make further specifications easier
This keyword expects a keyword. Default value: COEFFICIENTS
valid keywords:
  • ALL: Set all parameters in the basis to be variable.
  • NONE: Set all parameters in the basis to be fixed.
  • COEFFICIENTS: Set all coefficients in the basis set to be variable.
  • EXPONENTS: Set all exponents in the basis to be variable.
SWITCH_COEFF_STATE
SWITCH_COEFF_STATE SET L CONTRACTION IPGF
Allows to switch the state of a given coefficient from current state (varibale/fixed)) to the opposite state. The three integers indicate the set number, the angular momentum i'th contraction and i'th coefficient
This keyword expects precisely 4 integers, and may repeat
SWITCH_CONTRACTION_STATE
SWITCH_CONTRACTION_STATE SET L CONTRACTION
Allows to switch the state of a given contraction from current state (varibale/fixed)) to the opposite state. The three integers indicate the set number, the angular momentum and i'th contraction
This keyword expects precisely 3 integers, and may repeat
SWITCH_EXP_STATE
SWITCH_EXP_STATE SET IEXP
Allows to switch the state of a given exponent from current state (varibale/fixed)) to the opposite state. The two integers indicate the set number and i'th exponent
This keyword expects precisely 2 integers, and may repeat
SWITCH_SET_STATE
SWITCH_SET_STATE SET IEXP
Allows to switch the states of in a set from current state (varibale/fixed)) to the opposite state. The two integers indicate the affected part (0=ALL,1=EXPS,2=COEFF) and i'th set
This keyword expects precisely 2 integers, and may repeat
