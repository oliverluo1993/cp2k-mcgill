Section XC

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%KG_METHOD%ENERGY_CORRECTION%XC.

Keyword descriptions

DENSITY_CUTOFF
density_cutoff 1.e-11
The cutoff on the density used by the xc calculation
This keyword expects a real. Default value: 1.0000000000000000E-10
GRADIENT_CUTOFF
gradient_cutoff 1.e-11
The cutoff on the gradient of the density used by the xc calculation
This keyword expects a real. Default value: 1.0000000000000000E-10
DENSITY_SMOOTH_CUTOFF_RANGE
gradient_cutoff {real}
Parameter for the smoothing procedure inxc calculation
This keyword expects a real. Default value: 0.0000000000000000E+00
TAU_CUTOFF
tau_cutoff 1.e-11
The cutoff on tau used by the xc calculation
This keyword expects a real. Default value: 1.0000000000000000E-10
FUNCTIONAL_ROUTINE
FUNCTIONAL_ROUTINE (NEW|OLD|TEST_LSD|DEBUG)
Select the code for xc calculation
This keyword expects a keyword. Default value: NEW
valid keywords:
  • NEW: Use new code for exchange-correlation functional calculation
  • OLD: Use old code for exchange-correlation functional calculation
  • TEST_LSD: Use test local-spin-density approximation code for exchange-correlation functional calculation
  • DEBUG: Use debug new code for exchange-correlation functional calculation
