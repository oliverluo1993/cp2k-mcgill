Keyword descriptions

DIELECTRIC_CORE_CORRECTION
DIELECTRIC_CORE_CORRECTION
Avoid spurious values of the dielectric constant at the ionic core for pseudopotentials where the electron density goes to zero at the core (e.g. GTH). The correction is based on rho_core.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
DIELECTRIC_FUNCTION_TYPE
DIELECTRIC_FUNCTION_TYPE
Preferred type for the dielectric constant function.
This keyword expects a keyword. Default value: DENSITY_DEPENDENT
valid keywords:
  • DENSITY_DEPENDENT: Dielectric constant as a function of the electron density as e.g. proposed within the SCCS model.
  • SPATIALLY_DEPENDENT: Various regions with different dielectric constants.
  • SPATIALLY_RHO_DEPENDENT: Various regions with different dielectric constants. The dielectric constant decays to 1.0, wherever the electron density is present.
DIELECTRIC_CONSTANT
dielectric_constant
Dielectric constant in the bulk of the solvent.
This keyword expects a real. Default value: 8.0000000000000000E+01
variants: EPSILON
RHO_MIN
rho_min
Lower density threshold.
This keyword expects a real. Default value: 1.0000000000000000E-04
RHO_MAX
rho_max
Upper density threshold.
This keyword expects a real. Default value: 1.0000000000000000E-03
DERIVATIVE_METHOD
DERIVATIVE_METHOD
Preferred method for evaluating the gradient of ln(eps).
This keyword expects a keyword. Default value: CD7
valid keywords:
  • FFT: FFT based deriv of epsilon, without correction (high cutoff needed).
  • FFT_USE_DEPS: FFT based deriv of epsilon, with correction using gradient of epsilon (high cutoff needed).
  • FFT_USE_DRHO: FFT based deriv of epsilon, with correction using gradient of rho (high cutoff needed).
  • CD3: 3-point central difference derivative.
  • CD5: 5-point central difference derivative.
  • CD7: 7-point central difference derivative (recommended).
