Section CONSTRAIN_EXPONENTS

Index of all sections. This section is located at __ROOT__%OPTIMIZE_BASIS%FIT_KIND%CONSTRAIN_EXPONENTS.

specicifies contraints for the exponents to be fitted. Only a single constraint can be applied to an exponent. This section can be repeated.

Subsections

None

Section keywords

  • USE_EXP
  • BOUNDARIES
  • MAX_VAR_FRACTION

Keyword descriptions

USE_EXP
USE_EXP SET IEXP
Defines the exponent to be constraint. The two integers indicate the set number and i'th exponent. The value -1 can be used to mark all sets/expoenents in a set.
This keyword expects precisely 2 integers
BOUNDARIES
BOUNDARIES LOWER UPPER
Defines the boundaries to which the optimization is restricted. First value is the lower bound, second value is the upper bound.
This keyword expects precisely 2 reals
MAX_VAR_FRACTION
MAX_VAR_FRACTION REAL
Defines the maximum fractionr by which the exponent is allowed to vary. e.g. 0.5 allows the exp to vary by 0.5*exp in both directions.
This keyword expects a real
