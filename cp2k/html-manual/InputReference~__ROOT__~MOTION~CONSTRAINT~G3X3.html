The CP2K project : input section G3X3

Section G3X3

Index of all sections. This section is located at __ROOT__%MOTION%CONSTRAINT%G3X3.

This section is used to set 3x3 (3 atoms and 3 distances) constraints. This section can be repeated.

Subsections

  • RESTRAINT

Keyword descriptions

MOLECULE
MOL {integer}
Specifies the molecule kind number on which constraint will be applied. MOLECULE and MOLNAME keyword exclude themself mutually.
This keyword expects an integer
variants: MOL
MOLNAME
MOLNAME {character}
Specifies the name of the molecule on which the constraint will be applied.
This keyword expects a word
variants: SEGNAME
INTERMOLECULAR
INTERMOLECULAR
Specify if the constraint/restraint is intermolecular.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
ATOMS
ATOMS 1 3 6
Atoms' index on which apply the constraint
This keyword expects a list of integers
DISTANCES
DISTANCES {real} {real} {real}
The constrained distances' values.
This keyword expects a list of reals. Default unit: [internal_cp2k].
EXCLUDE_QM
EXCLUDE_QM
Does not apply the constraint to the QM region within a QM/MM calculation
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
EXCLUDE_MM
EXCLUDE_MM
Does not apply the constraint to the MM region within a QM/MM calculation
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
