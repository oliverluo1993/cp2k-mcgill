Section ENFORCE_OCCUPATION

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%KIND%DFT_PLUS_U%ENFORCE_OCCUPATION.

Enforce and control a special (initial) orbital occupation. Note, this feature works only for the methods MULLIKEN and LOWDIN. It should only be used to prepare an initial configuration. An inadequate parameter choice can easily inhibit SCF convergence. This section can not be repeated.
