Section TEMP_CONTROL

Index of all sections. This section is located at __ROOT__%MOTION%BAND%OPTIMIZE_BAND%MD%TEMP_CONTROL.

Setup parameters to control the temperature during a BAND MD run. This section can not be repeated.

Subsections

None

Section keywords

  • TEMPERATURE
  • TEMP_TOL
  • TEMP_TOL_STEPS

Keyword descriptions

TEMPERATURE
Specify the target temperature
This keyword expects a real. Default unit: [K].
TEMP_TOL
Specify the tolerance on the temperature for rescaling
This keyword expects a real. Default unit: [K], default value: 0.0000000000000000E+00 K
TEMP_TOL_STEPS
Specify the number of steps to apply a temperature control
This keyword expects an integer. Default value: 0
