Section REPLICA

Index of all sections. This section is located at __ROOT__%MOTION%BAND%REPLICA.

Specify coordinates and velocities (possibly) of the replica. This section can be repeated.

Keyword descriptions

COLLECTIVE
COLLECTIVE {real} .. {real}
Specifies the value of the collective variables used in the projected BAND method. The order of the values is the order of the COLLECTIVE section in the constraints/restraints section
This keyword expects a list of reals
