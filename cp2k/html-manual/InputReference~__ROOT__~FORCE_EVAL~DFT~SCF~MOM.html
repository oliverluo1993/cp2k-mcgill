Section MOM

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%SCF%MOM.

Define type and parameters for the maximum overlap method (MOM) to determine orbital occupancies. The MOM procedures activated by this section are only active for diagonalization methods, i.e. not with minimization methods based on OT. Incompatible with core-level excitation spectra (XAS). This section can not be repeated.

This section cites following reference: [Gilbert2008]

This keyword behaves as a switch T . Default value: F
START_ITER

SCF iteration cycle to start the MOM procedure. Could be used for ground state calculations only in order to stabilise oscillating SCF iterations
This keyword expects an integer. Default value: 0

PROJ_FORMULA
SCF_GUESS RESTART
Projection formula to be used
This keyword expects a keyword. Default value: NORM
valid keywords:
  • NORM: The one which ignores the phase of molecular orbitals: proj_j = \sqrt(\sum_i overlap(i, j)^2)
  • SUM: The one proposed in the original paper: proj_j = |\sum_i overlap(i, j)|
