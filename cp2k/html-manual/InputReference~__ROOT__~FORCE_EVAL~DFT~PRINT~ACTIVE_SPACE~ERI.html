Keyword descriptions

METHOD
METHOD FULL_GPW
Method used in ERI calculation.
This keyword expects a keyword. Default value: GPW_HALF_TRANSFORM
valid keywords:
  • FULL_GPW: Use the GPW approach with MOs
  • GPW_HALF_TRANSFORM: Use the GPW approach for half-transformed MO ERIs
OPERATOR
OPERATOR <1/R>
Operator used in ERI calculation.
This keyword expects a keyword. Default value: <1/R>
valid keywords:
  • <1/R>: Coulomb operator
  • : Yukawa potential operator
  • : Error function potential operator
  • : Complementary error function potential operator
  • : Gaussian potential operator
OPERATOR_PARAMETER
OPERATOR_PARAMETER 4.0
Range parameter for ERI operator.
This keyword expects a real. Default value: 8.0000000000000000E+00
PERIODICITY
PERIODICITY {1 1 1}
Periodicity used for operators in ERI calclulation.
This keyword expects a list of integers. Default value: 1 1 1
CUTOFF_RADIUS
CUTOFF_RADIUS 20
Radius of operator interactions in non-periodic cases.
This keyword expects a real. Default value: -1.0000000000000000E+00
EPS_INTEGRAL
EPS_FILTER 1.0E-10
Accuracy of ERIs that will be stored.
This keyword expects a real. Default value: 9.9999999999999998E-13
