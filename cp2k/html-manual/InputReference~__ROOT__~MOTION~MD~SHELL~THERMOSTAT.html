Section THERMOSTAT

Index of all sections. This section is located at __ROOT__%MOTION%MD%SHELL%THERMOSTAT.

Specify thermostat type and parameters controlling the thermostat. This section can not be repeated.

Keyword descriptions

TYPE
thermostat NOSE
Specify the thermostat used for the constant temperature ensembles.
This keyword expects a keyword. Default value: NOSE
valid keywords:
  • NOSE: Uses the Nose-Hoover thermostat.
  • CSVR: Uses the canonical sampling through velocity rescaling.
  • GLE: Uses GLE thermostat
  • AD_LANGEVIN: Uses adaptive-Langevin thermostat
REGION
REGION (GLOBAL||MOLECULE||MASSIVE||DEFINED||NONE)
Determines the region each thermostat is attached to.
This keyword expects a keyword. Default value: GLOBAL
valid keywords: GLOBAL, MOLECULE, MASSIVE, DEFINED, NONE
