Section PRINT_ORBITAL_CUBES

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%PRINT%ACTIVE_SPACE%PRINT_ORBITAL_CUBES.

Controls printing of active orbital cube files. This section can not be repeated.

Subsections

None

Section keywords

  • FILENAME
  • ALIST
  • BLIST
  • STRIDE

Keyword descriptions

FILENAME
FILENAME {name}
Body of Filename for the cube files.
This keyword expects a word. Default value: ActiveOrbital
ALIST
ALIST {1 2 3 ...}
List of alpha orbitals to be printed. -1 defaults to all values
This keyword expects a list of integers
This keyword behaves as a switch -1 . Default value: -1
BLIST
ALIST {1 2 3 ...}
List of beta orbitals to be printed. -1 defaults to all values
This keyword expects a list of integers
This keyword behaves as a switch -1 . Default value: -1
STRIDE
STRIDE {2 2 2}
The stride (X,Y,Z) used to write the cube file(larger values result in smaller cube files). You can provide 3 numbers (for X,Y,Z) or 1 number valid for all components
This keyword expects a list of integers. Default value: 2 2 2
