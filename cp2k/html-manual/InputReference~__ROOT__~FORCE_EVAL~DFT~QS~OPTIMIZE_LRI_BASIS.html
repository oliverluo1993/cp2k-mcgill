Section OPTIMIZE_LRI_BASIS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%QS%OPTIMIZE_LRI_BASIS.

This section specifies the parameters for optimizing the lri auxiliary basis sets for LRIGPW. The Powell optimizer is used. This section can not be repeated.

Keyword descriptions

ACCURACY
ACCURACY 5.0E-4
Target accuracy for the objective function (RHOEND)
This keyword expects a real. Default value: 1.0000000000000001E-05
MAX_FUN
MAX_FUN 200
Maximum number of function evaluations
This keyword expects an integer. Default value: 4000
STEP_SIZE
STEP_SIZE 1.0E-1
Initial step size for search algorithm (RHOBEG)
This keyword expects a real. Default value: 5.0000000000000003E-02
CONDITION_WEIGHT
CONDITION_WEIGHT 1.0E-4
This keyword allows to give different weight factors to the condition number (LOG(cond) is used).
This keyword expects a real. Default value: 9.9999999999999995E-07
USE_CONDITION_NUMBER
USE_CONDITION_NUMBER
Determines whether condition number should be part of optimization or not
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
GEOMETRIC_SEQUENCE
GEOMETRIC_SEQUENCE
Exponents are assumed to be a geometric squence. Only the minimal and maximal exponents of one set are optimized and the other exponents are obtained by geometric progression.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
DEGREES_OF_FREEDOM
DEGREES_OF_FREEDOM ALL
Specifies the degrees of freedom in the basis optimization.
This keyword expects a keyword. Default value: EXPONENTS
valid keywords:
  • ALL: Set all parameters in the basis to be variable.
  • COEFFICIENTS: Set all coefficients in the basis set to be variable.
  • EXPONENTS: Set all exponents in the basis to be variable.
