Section HYDRONIUM_SHELL

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%COLVAR%HYDRONIUM_SHELL.

Section to define the formation of a hydronium as a collective variable. Number of oxygens in the 1st shell of the hydronium. Adapted from Equation (3) in Supplementary Info of J. Am. Chem. Soc.,128, 2006, 11318, i.e. omitting the cutoff function and summing only over the oxygens of water. This section can not be repeated.

Keyword descriptions

OXYGENS
OXYGENS {integer} {integer} ..
Specifies indexes of atoms building the coordination variable. Oxygens of the water molecules.
This keyword expects a list of integers, and may repeat
HYDROGENS
HYDROGENS {integer} {integer} ..
Specifies indexes of atoms building the coordination variable. Hydrogens of the water molecules.
This keyword expects a list of integers, and may repeat
ROO
ROO {real}
Specifies the rc parameter in the coordination function: number of oxygens per water oxygen.
This keyword expects a real. Default unit: [bohr], default value: 3.0000000000000000E+00 bohr
POO
pOO {integer}
Sets the value of the numerator of the exponential factor in the coordination function: number of oxygens per water oxygen.
This keyword expects an integer. Default value: 6
variants: EXPON_NUMERATORA
QOO
qOO {integer}
Sets the value of the denominator of the exponential factor in the coordination function: number of oxygens per water oxygen.
This keyword expects an integer. Default value: 12
variants: EXPON_DENOMINATORA
ROH
ROH {real}
Specifies the rc parameter in the coordination function: number of hydrogens per water molecule.
This keyword expects a real. Default unit: [bohr], default value: 3.0000000000000000E+00 bohr
POH
pOH {integer}

This keyword expects an integer. Default value: 6
variants: EXPON_NUMERATORB
QOH

variants: EXPON_DENOMINATORB
