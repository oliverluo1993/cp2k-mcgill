Section DUMMY_ATOMS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%PROPERTIES%ET_COUPLING%BECKE_CONSTRAINT_B%DUMMY_ATOMS.

Define an extra group of atoms for which only atomic Becke charges should be computed. The section cannot contain any constraintatoms which should be included in section ATOM_GROUP. This section can be repeated.
