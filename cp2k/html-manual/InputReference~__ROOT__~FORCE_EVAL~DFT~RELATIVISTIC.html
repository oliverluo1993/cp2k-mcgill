Section RELATIVISTIC

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%RELATIVISTIC.

parameters needed and setup for relativistic calculations. This section can not be repeated.

Subsections

None

Keyword descriptions

METHOD
method (NONE|DKH|ZORA)
type of relativistic correction used
This keyword expects a keyword. Default value: NONE
valid keywords:
  • NONE: Use no relativistic correction
  • DKH: Use Douglas-Kroll-Hess method
  • ZORA: Use ZORA method
DKH_ORDER
DKH_order 2
The order of the DKH transformation
This keyword expects an integer. Default value: 2
ZORA_TYPE
ZORA_type scMP
Type of ZORA method to be used
This keyword expects a keyword. Default value: FULL
valid keywords:
  • FULL: Full ZORA method (not implemented)
  • MP: ZORA with atomic model potential
  • SCMP: Scaled ZORA with atomic model potential
TRANSFORMATION
transformation (FULL|MOLECULE|ATOM)
Type of DKH transformation
This keyword expects a keyword. Default value: FULL
valid keywords:
  • FULL: Use full matrix transformation
  • MOLECULE: Use transformation blocked by molecule
  • ATOM: Use atomic blocks
Z_CUTOFF
z_cutoff 50
The minimal atomic number considered for atom transformation
This keyword expects an integer. Default value: 1
POTENTIAL
POTENTIAL {FULL,ERFC}
External potential used in DKH transformation, full 1/r or erfc(r)/r
This keyword expects a keyword. Default value: ERFC
valid keywords: FULL, ERFC
