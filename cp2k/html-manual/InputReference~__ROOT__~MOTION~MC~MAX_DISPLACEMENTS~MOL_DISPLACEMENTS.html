Section MOL_DISPLACEMENTS

Index of all sections. This section is located at __ROOT__%MOTION%MC%MAX_DISPLACEMENTS%MOL_DISPLACEMENTS.

Maximum displacements for every move type that requires a value for each molecular type in the simulation. This section can not be repeated.

Subsections

None

Keyword descriptions

RMBOND
RMBOND {real} {real} ...
Maximum bond length displacement, in angstroms, for each molecule type.
This keyword expects a list of reals
