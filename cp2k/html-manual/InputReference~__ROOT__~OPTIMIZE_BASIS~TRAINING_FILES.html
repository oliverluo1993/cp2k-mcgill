Section TRAINING_FILES

Index of all sections. This section is located at __ROOT__%OPTIMIZE_BASIS%TRAINING_FILES.

specicifies the location in which the files necessary for fitting procedure are located. Each Training set needs a reptition of this section. This section can be repeated.

Subsections

None

Section keywords

Keyword descriptions

DIRECTORY
DIRECTORY /my/path
the directory in which the files are placed
This keyword expects a string. Default value: .
INPUT_FILE_NAME
INPUT_FILE_NAME my_input.inp
the filename of the input file used to run the original calcuation
This keyword expects a string. Default value: input.inp
