Section SHELL_VELOCITY

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%SHELL_VELOCITY.

The velocities of shells for shell-model potentials, in xyz format. This section can not be repeated.

Keyword descriptions

_DEFAULT_KEYWORD_
{Real} {Real} {Real}
The shell particle velocities in the format:

vx vy vz

The same order as for the shell particle coordinates is assumed.
This keyword expects precisely 3 reals, and may repeat
