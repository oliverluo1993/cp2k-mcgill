Section DERIVED_BASIS_SETS

Index of all sections. This section is located at __ROOT__%OPTIMIZE_BASIS%FIT_KIND%DERIVED_BASIS_SETS.

This section can be used to create subsets of a basis which will be fitted at the same time. This is especially useful if connected bsis sets e.g. TZVP, DZVP, SZV should be fitted. This section can be repeated.

Subsections

None

Section keywords

  • BASIS_SET_NAME
  • REFERENCE_SET
  • REMOVE_CONTRACTION
  • REMOVE_SET

Keyword descriptions

BASIS_SET_NAME
BASIS_SET_NAME {word}
Defines the name of the derived basis set, which will be automatically generated otherwise.
This keyword expects a word. Default value: " "
REFERENCE_SET
REFERNCE_SET INTEGER
Specifies the reference basis ID which is used as template to create the new set. The original basis has ID 0. All follwing sets are counted in order as specified in the Input. The decriptors always assume the structure of the input basis set.
This keyword expects an integer. Default value: 0
REMOVE_CONTRACTION
REMOVE_CONTRACTION SET L ICONTRACTION
Can be used to remove a contraction from the reference basis set. The contraction is speciefied by set number, angular momentum and number of contraction. The decriptors always assume the structure of the input basis set.
This keyword expects precisely 3 integers, and may repeat
REMOVE_SET
REMOVE_SET SET
Can be used to remove a set from the reference basis set.
This keyword expects an integer, and may repeat
