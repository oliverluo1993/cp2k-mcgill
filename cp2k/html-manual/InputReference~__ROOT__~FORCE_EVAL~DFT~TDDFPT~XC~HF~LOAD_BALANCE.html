Keyword descriptions

NBINS
NBINS 32
Number of bins per process used to group atom quartets.
This keyword expects an integer. Default value: 64
BLOCK_SIZE
BLOCK_SIZE 4
Determines the blocking used for the atomic quartet loops. A proper choice can speedup the calculation. The default (-1) is automatic.
This keyword expects an integer. Default value: -1
RANDOMIZE
RANDOMIZE TRUE
This flag controls the randomization of the bin assignment to processes. For highly ordered input structures with a bad load balance, setting this flag to TRUE might improve.
This keyword expects a logical. Default value: F
