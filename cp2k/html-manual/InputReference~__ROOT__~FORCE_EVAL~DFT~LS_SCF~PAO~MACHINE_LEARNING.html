Keyword descriptions

METHOD
Machine learning scheme used to predict PAO basis sets.
This keyword expects a keyword. Default value: GAUSSIAN_PROCESS
valid keywords:
  • GAUSSIAN_PROCESS: Gaussian Process
  • NEURAL_NETWORK: Neural Network
  • LAZY: Just rely on prior
PRIOR
Prior used for predictions.
This keyword expects a keyword. Default value: ZERO
valid keywords:
  • ZERO: Simply use zero
  • MEAN: Use average of training-set
DESCRIPTOR
Descriptor used as input for machine learning.
This keyword expects a keyword. Default value: POTENTIAL
valid keywords:
  • POTENTIAL: Eigenvalues of local potential matrix
  • OVERLAP: Eigenvalues of local overlap matrix
  • R12: Distance between two atoms (just for testing)

This keyword cites following reference: [Zhu2016]
TOLERANCE
Maximum variance tolerated when making predictions.
This keyword expects a real. Default value: 1.0000000000000000E-02
GP_NOISE_VAR
Variance of noise used for Gaussian Process machine learning.
This keyword expects a real. Default value: 1.0000000000000001E-01
GP_SCALE
Length scale used for Gaussian Process machine learning.
This keyword expects a real. Default value: 5.0000000000000003E-02
