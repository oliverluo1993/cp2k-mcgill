Section PLANAR

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%POISSON%IMPLICIT%DIRICHLET_BC%PLANAR.

Parameters for creating planar (rectangular) Dirichlet boundary regions with given vertices. This section can be repeated.

Keyword descriptions

V_D
v_D
The value of the fixed potential to be imposed at the planar Dirichlet boundary.
This keyword expects a real. Default unit: [volt].
OSCILLATING_FRACTION
OSCILLATING_FRACTION
A fraction of the field can be set to oscilate over time.
This keyword expects a real. Default value: 0.0000000000000000E+00
FREQUENCY
FREQUENCY
The frequency with which the oscillating fraction oscillates.
This keyword expects a real. Default unit: [s^-1], default value: 0.0000000000000000E+00 s^-1
PHASE
PHASE
The phase of the oscillattion. A phase of zero corresponds to a cosine function.
This keyword expects a real. Default value: 0.0000000000000000E+00
A
A
Coordinates of the vertex A.
This keyword expects precisely 3 reals. Default unit: [angstrom].
B
B
Coordinates of the vertex B.
This keyword expects precisely 3 reals. Default unit: [angstrom].
C
C
Coordinates of the vertex C.
This keyword expects precisely 3 reals. Default unit: [angstrom].
N_PRTN
N_PRTN
The number of partitions along the edges for tiling the rectangular region. If the edges have different lengths, from the two given values, the larger one will be assigned to the longer edge.
This keyword expects precisely 2 integers. Default value: 1 1
THICKNESS
THICKNESS
The thickness of the planar Dirichlet region.
This keyword expects a real. Default unit: [angstrom], default value: 3.9688290644250002E-01 angstrom
SMOOTHING_WIDTH
SMOOTHING_WIDTH
The width of the transition region between the Dirichlet subdomain and its surrounding.
This keyword expects a real. Default unit: [angstrom], default value: 1.0583544171800002E-01 angstrom
variants: SIGMA
