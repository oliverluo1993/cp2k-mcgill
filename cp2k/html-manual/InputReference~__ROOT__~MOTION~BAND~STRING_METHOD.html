Keyword descriptions

SPLINE_ORDER
Specify the oder of the spline used in the String Method.
This keyword expects an integer. Default value: 1
SMOOTHING
Smoothing parameter for the reparametrization of the frames.
This keyword expects a real. Default value: 2.0000000000000001E-01
