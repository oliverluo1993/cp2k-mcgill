Section AVBMC

Index of all sections. This section is located at __ROOT__%MOTION%MC%AVBMC.

Parameters for Aggregation Volume Bias Monte Carlo (AVBMC) which explores cluster formation and destruction. Chen and Siepmann, J. Phys. Chem. B 105, 11275-11282 (2001). This section can not be repeated.

Subsections

None

Section keywords

  • AVBMC_ATOM

Keyword descriptions

PBIAS
PBIAS {real} {real} ...
The probability of swapping to an inner region in an AVBMC swap move for each molecule type.
This keyword expects a list of reals
AVBMC_ATOM
AVBMC_ATOM {integer} {integer} ...
The target atom for an AVBMC swap move for each molecule type.
This keyword expects a list of integers
AVBMC_RMIN
AVBMC_RMIN {real} {real} ...
The inner radius for an AVBMC swap move, in angstroms for every molecule type.
This keyword expects a list of reals
AVBMC_RMAX
AVBMC_RMAX {real} {real} ...
The outer radius for an AVBMC swap move, in angstroms, for every molecule type.
This keyword expects a list of reals
