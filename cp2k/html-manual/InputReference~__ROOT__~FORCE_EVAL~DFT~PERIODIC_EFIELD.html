Section PERIODIC_EFIELD

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%PERIODIC_EFIELD.

parameters for finite periodic electric field computed using the Berry phase approach. IMPORTANT: Can only be used in combination with OT. Can not be used in combination with RTP or EMD. This section can be repeated.

This section cites following references: [Souza2002] [Umari2002]
