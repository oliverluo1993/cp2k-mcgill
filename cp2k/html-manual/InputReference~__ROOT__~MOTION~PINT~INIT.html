The CP2K project : input section INIT

Section INIT

Index of all sections. This section is located at __ROOT__%MOTION%PINT%INIT.

Controls the initialization if the beads are not present. This section can not be repeated.

Subsections

None

Keyword descriptions

LEVY_POS_SAMPLE
Sample bead positions assuming free particle behavior (performs a Levy random walk of length P around the classical position of each atom at the physical temperature defined in PINT%TEMP)
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
LEVY_CORRELATED
Use the same Levy path for all atoms, though with mass-dependent variances (might help at very low T)
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
LEVY_TEMP_FACTOR
Multiplicative correction factor for the temperature at which the Levy walk is performed (correction is due to the interactions that modify the spread of a free particle)
This keyword expects a real. Default value: 1.0000000000000000E+00
LEVY_SEED
LEVY_SEED
Initial seed for the (pseudo)random number generator that controls Levy walk for bead positions.
This keyword expects an integer. Default value: 1234
THERMOSTAT_SEED
THERMOSTAT_SEED
Initial seed for the (pseudo)random number generator that controls the PILE and PIGLET thermostats.
This keyword expects an integer. Default value: 12345
RANDOMIZE_POS
add gaussian noise to the positions of the beads
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
CENTROID_SPEED
adds random velocity component to the centroid modes (useful to correct for the averaging out of the speed of various beads)
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
VELOCITY_QUENCH
set the initial velocities to zero
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
VELOCITY_SCALE
scale initial velocities to the temperature given in MOTION%PINT%TEMP
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
