Section CENTER

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%SCRF%SPHERE%CENTER.

Defines the center of the sphere. This section can not be repeated.

None

Keyword descriptions

XYZ
XYZ
Coordinates of the center of the sphere
This keyword expects precisely 3 reals. Default unit: [angstrom].
ATOM_LIST
ATOM_LIST ..
Defines a list of atoms to define the center of the sphere
This keyword expects a list of integers
WEIGHT_TYPE
WEIGHT (UNIT|MASS)
Defines the weight used to define the center of the sphere (if ATOM_LIST is provided)
This keyword expects a keyword. Default value: UNIT
valid keywords: UNIT, MASS
FIXED
FIXED
Specify if the center of the sphere should be fixed or allowed to move
This keyword expects a logical. Default value: T
