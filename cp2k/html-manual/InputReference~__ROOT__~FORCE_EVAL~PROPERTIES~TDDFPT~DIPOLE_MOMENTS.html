Section DIPOLE_MOMENTS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%PROPERTIES%TDDFPT%DIPOLE_MOMENTS.

Parameters to compute oscillator strengths in the dipole approximation. This section can not be repeated.

Subsections

None

Section keywords

  • DIPOLE_FORM
  • REFERENCE
  • REFERENCE_POINT

Keyword descriptions

DIPOLE_FORM
Form of dipole transition integrals.
This keyword expects a keyword. Default value: VELOCITY
valid keywords:
  • LENGTH: Length form ⟨ i | r | j ⟩
  • VELOCITY: Velocity form ⟨ i | d/dr | j ⟩
REFERENCE
Reference point to calculate electric dipole moments using the dipole integrals in the length form.
This keyword expects a keyword. Default value: COM
valid keywords:
  • COM: Use Center of Mass
  • COAC: Use Center of Atomic Charges
  • USER_DEFINED: Use User-defined Point
  • ZERO: Use Origin of Coordinate System
REFERENCE_POINT
REFERENCE_POINT x y z
User-defined reference point.
This keyword expects precisely 3 reals. Default unit: [bohr].
