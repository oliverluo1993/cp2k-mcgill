Section MD

Index of all sections. This section is located at __ROOT__%MOTION%BAND%OPTIMIZE_BAND%MD.

Activate the MD based optimization procedure for BAND. This section can not be repeated.

Subsections

  • TEMP_CONTROL
  • VEL_CONTROL

Section keywords

  • MAX_STEPS
  • TIMESTEP
  • TEMPERATURE

Keyword descriptions

MAX_STEPS
Specify the maximum number of MD steps
This keyword expects an integer. Default value: 100
TIMESTEP
timestep 1.0
The length of an integration step
This keyword expects a real. Default unit: [fs], default value: 4.9999999999999994E-01 fs
TEMPERATURE
Specify the initial temperature
This keyword expects a real. Default unit: [K], default value: 0.0000000000000000E+00 K
