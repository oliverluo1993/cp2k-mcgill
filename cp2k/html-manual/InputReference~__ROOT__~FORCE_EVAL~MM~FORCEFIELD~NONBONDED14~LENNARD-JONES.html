Section LENNARD-JONES

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%MM%FORCEFIELD%NONBONDED14%LENNARD-JONES.

This section specifies the input parameters for LENNARD-JONES potential type.Functional form: V(r) = 4.0 * EPSILON * [(SIGMA/r)^12-(SIGMA/r)^6]. This section can be repeated.
