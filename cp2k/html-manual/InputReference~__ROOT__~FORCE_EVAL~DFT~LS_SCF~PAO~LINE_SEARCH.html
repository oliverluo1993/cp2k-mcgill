Keyword descriptions

METHOD
Linesearch method.
This keyword expects a keyword. Default value: ADAPT
valid keywords:
  • ADAPT: extrapolates usually based on 3 points, uses additional points on demand, very robust.
  • 3PNT: extrapolate based on 3 points
  • 2PNT: extrapolate based on 2 points and the slope (super fast, but might get stuck at saddle points)
  • GOLD: perform 1D golden section search of the minimum (very expensive)
  • NONE: always take steps of fixed INITIAL_STEP_SIZE
INITIAL_STEP_SIZE
Initial step length
This keyword expects a real. Default value: 1.0000000000000001E-01
MAX_STEP_SIZE
Maximum step length
This keyword expects a real. Default value: 3.0000000000000000E+00
TINY_STEP_SIZE
Step length taken if negative step is suggested.
This keyword expects a real. Default value: 2.0000000000000000E-03
EPS_STEP_SIZE
Convergence criterion of GOLD method.
This keyword expects a real. Default value: 1.0000000000000001E-01
