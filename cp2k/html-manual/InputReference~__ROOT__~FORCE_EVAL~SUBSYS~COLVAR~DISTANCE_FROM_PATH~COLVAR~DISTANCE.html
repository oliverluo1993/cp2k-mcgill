Section DISTANCE

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%COLVAR%DISTANCE_FROM_PATH%COLVAR%DISTANCE.

Section to define the distance as a collective variables. This section can not be repeated.

AXIS

This keyword expects a keyword. Default value: XYZ
