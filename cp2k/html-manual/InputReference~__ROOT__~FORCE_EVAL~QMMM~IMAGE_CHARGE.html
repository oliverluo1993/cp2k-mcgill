Section IMAGE_CHARGE

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%QMMM%IMAGE_CHARGE.

Inclusion of polarization effects within the image charge approach for systems where QM molecules are physisorbed on e.g. metal surfaces described by MM. This correction introduces only a very small overhead. QM box size has to be equal to MM box size. This section can not be repeated.

This section cites following reference: [Golze2013]

Keyword descriptions

MM_ATOM_LIST
MM_ATOM_LIST 1 2 3 or 1..3
List of MM atoms carrying an induced Gaussian charge. If this keyword is not given, all MM atoms will carry an image charge.
This keyword expects a list of integers, and may repeat
WIDTH
WIDTH
Specifies the value of the width of the (induced) Gaussian charge distribution carried by each MM atom.
This keyword expects a real. Default unit: [angstrom^-2], default value: 3.0000000000000000E+00 angstrom^-2
EXT_POTENTIAL
EXT_POTENTIAL
External potential applied to the metal electrode
This keyword expects a real. Default unit: [volt], default value: 0.0000000000000000E+00 volt
DETERM_COEFF
DETERM_COEFF ITERATIVE
Specifies how the coefficients are determined.
This keyword expects a keyword. Default value: CALC_MATRIX
valid keywords:
  • CALC_MATRIX: Calculates image matrix and solves linear set of equations
  • ITERATIVE: Uses an iterative scheme to calculate the coefficients
RESTART_IMAGE_MATRIX
RESTART_IMAGE_MATRIX
Restart the image matrix. Useful when calculating coefficients iteratively (the image matrix is used as preconditioner in that case)
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
IMAGE_RESTART_FILE_NAME
IMAGE_RESTART_FILE_NAME
File name where to read the image matrix used as preconditioner in the iterative scheme
This keyword expects a string
IMAGE_MATRIX_METHOD
IMAGE_MATRIX_METHOD MME
Method for calculating the image matrix.
This keyword expects a keyword. Default value: MME
valid keywords:
  • GPW: Uses Gaussian Plane Wave method [Golze2013]
  • MME: Uses MiniMax-Ewald method (ERI_MME subsection)
