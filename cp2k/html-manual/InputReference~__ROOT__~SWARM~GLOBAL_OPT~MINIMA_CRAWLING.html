Section MINIMA_CRAWLING

Index of all sections. This section is located at __ROOT__%SWARM%GLOBAL_OPT%MINIMA_CRAWLING.

Section controls Minima Crawling run. This section can not be repeated.

Subsections

  • MINIMA_TRAJECTORY

Keyword descriptions

TEMPSTEP_BASE
Base used to calculate temperature steps base**n
This keyword expects a real. Default value: 1.1000000000000001E+00
TEMPSTEP_MAX
Maximum number of temperature steps.
This keyword expects an integer. Default value: 100
TEMPDIST_UPDATE_WIDTH
Width of gaussian used to update temperature distribution.
This keyword expects a real. Default value: 2.0000000000000000E+00
TEMPDIST_UPDATE_HEIGHT
Height of gaussian used to update temperature distribution.
This keyword expects a real. Default value: 1.0000000000000001E-01
TEMPERATURE_INIT
Initial temperature in Kelvin
This keyword expects a real. Default value: 1.0000000000000000E+02
TEMPDIST_INIT_WIDTH
Initial width of temperature distribution.
This keyword expects a real. Default value: 5.0000000000000000E+00
WORKER_PER_MINIMA
Maximum number of active workers per Minima.
This keyword expects an integer. Default value: 3
ESCAPE_HISTORY_LENGTH
Number of escapes averaged for scoring of minima.
This keyword expects an integer. Default value: 10
