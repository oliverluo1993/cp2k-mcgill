Section ACTIVE_SPACE

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%PRINT%ACTIVE_SPACE.

Define parameters and method to calculate an electronic active space. This section can not be repeated.

Keyword descriptions

_SECTION_PARAMETERS_
&ACTIVE_SPACE ON
Controls the activation of the ACTIVE_SPACE section
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
MODEL
MODEL HARTREE-FOCK
Model used to calculate active space interaction Hamiltonian.
This keyword expects a keyword. Default value: HARTREE-FOCK
valid keywords:
  • HARTREE-FOCK: Hartree-Fock model for interaction Hamiltonian
  • RSDFT: Range-separated DFT model for interaction Hamiltonian
  • DMFT: DMFT model Hamiltonian
ACTIVE_ELECTRONS
ACTIVE_ELECTRONS 4
The number of active electrons in the CAS space
This keyword expects an integer. Default value: -1
INACTIVE_ELECTRONS
INACTIVE_ELECTRONS 2 {2}
The number of inactive electrons in the CAS space
This keyword expects a list of integers. Default value: -1 -1
ACTIVE_ORBITALS
ACTIVE_ORBITALS 2 {2}
The number of active orbitals defining the CAS space.
This keyword expects a list of integers. Default value: -1 -1
INACTIVE_ORBITALS
INACTIVE_ORBITALS 2 {2}
The number of nactive orbitals.
This keyword expects a list of integers. Default value: -1 -1
ISOLATED_SYSTEM
System is treated without any periodic boundary conditions.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
ORBITAL_SELECTION
ORBITAL_SELECTION CANONICAL
Method used to select active space orbitals.
This keyword expects a keyword. Default value: CANONICAL
valid keywords:
  • CANONICAL: Select orbitals using energy ordering of canoncial orbitals
  • WANNIER_PROJECTION: Select orbitals from projected Wannier functions
  • MAO: Select orbitals from modified atomic orbitals
SUBSPACE_ATOM
SUBSPACE_ATOM x
Number of atom that defines the subspace to be projected on.
This keyword expects an integer. Default value: -1
SUBSPACE_SHELL
SUBSPACE_SHELL 3d4s
Shell definition for subsapce.
This keyword expects a word. Default value: X
