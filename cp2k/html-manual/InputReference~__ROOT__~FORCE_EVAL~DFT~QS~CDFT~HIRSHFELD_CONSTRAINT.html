Section HIRSHFELD_CONSTRAINT

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%QS%CDFT%HIRSHFELD_CONSTRAINT.

Parameters for CDFT with Hirshfeld constraint. This section can not be repeated.

Keyword descriptions

STRENGTH
STRENGTH {real} {real}
Force constants of the constraints. Second value is for combined constraint and is optional.
This keyword expects a list of reals. Default value: 0.0000000000000000E+00
TARGET
TARGET {real}
Target values of the constraints. Second value is for combined constraint and is optional.
This keyword expects a list of reals. Default value: 0.0000000000000000E+00
ATOMS
ATOMS {integer} {integer} .. {integer}
Specifies the list of atoms that are included in the constraint
This keyword expects a list of integers
COEFF
COEFF 1.0 -1.0
Defines coefficients for atoms included in the constraint (default is one) Use +1.0 for donor atoms and -1.0 for acceptor atoms.
This keyword expects a list of reals, and may repeat
SELF_CONSISTENT
SELF_CONSISTENT yes
Calculate charges from the Hirsheld-I (self_consistent) method. This scales only the full shape function, not the added charge as in the original scheme. NYI.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
SHAPE_FUNCTION
SHAPE_FUNCTION {Gaussian,Density}
Type of shape function used for Hirshfeld partitioning.
This keyword expects a keyword. Default value: GAUSSIAN
valid keywords:
  • GAUSSIAN: Single Gaussian with Colvalent radius
  • DENSITY: Atomic density expanded in multiple Gaussians (NYI)
REFERENCE_CHARGE
REFERENCE_CHARGE {Atomic,Mulliken}
Charge of atomic partitioning function for Hirshfeld method.
This keyword expects a keyword. Default value: ATOMIC
valid keywords:
  • ATOMIC: Use atomic core charges
  • MULLIKEN: Calculate Mulliken charges (NYI)
USER_RADIUS
USER_RADIUS yes
Use user defined covalent radii for single Gaussian Hirshfeld partitioning. These radii are defined by the keyword ATOMIC_RADII
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
ATOMIC_RADII
ATOMIC_RADII {real} {real} {real}
Defines custom radii to setup the spherical Gaussians.
This keyword expects a list of reals. Default unit: [angstrom].
