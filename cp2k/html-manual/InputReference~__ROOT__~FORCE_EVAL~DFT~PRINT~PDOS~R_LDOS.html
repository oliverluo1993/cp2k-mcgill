Section R_LDOS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%PRINT%PDOS%R_LDOS.

Controls the printing of local PDOS, projected on 3D volume in real space, the volume is defined in terms of position with respect to atoms in the lists. This section can be repeated.

Subsections

None

Section keywords

  • LIST
  • XRANGE
  • YRANGE
  • ZRANGE
  • ERANGE

Keyword descriptions

LIST
LIST {integer} {integer} .. {integer}
Specifies a list of indexes of atoms used to define the real space volume
This keyword expects a list of integers, and may repeat
XRANGE
XRANGE -10.0 10.0
range of positions in Cartesian direction x: all grid points within this range from at least one atom of the list are considered
This keyword expects precisely 2 reals. Default unit: [angstrom].
YRANGE
YRANGE -10.0 10.0
range of positions in Cartesian direction y: all grid points within this range from at least one atom of the list are considered
This keyword expects precisely 2 reals. Default unit: [angstrom].
ZRANGE
ZRANGE -10.0 10.0
range of positions in Cartesian direction z: all grid points within this range from at least one atom of the list are considered
This keyword expects precisely 2 reals. Default unit: [angstrom].
ERANGE
ERANGE -1.0 1.0
only project states with the eigenvalues in the given interval. Default is all states.
This keyword expects precisely 2 reals. Default unit: [hartree].
