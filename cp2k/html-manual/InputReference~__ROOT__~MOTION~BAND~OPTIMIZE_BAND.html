Section OPTIMIZE_BAND

Index of all sections. This section is located at __ROOT__%MOTION%BAND%OPTIMIZE_BAND.

Specify the optimization method for the band. This section can be repeated.

Subsections

  • MD
  • DIIS

Section keywords

  • OPT_TYPE
  • OPTIMIZE_END_POINTS

Keyword descriptions

OPT_TYPE
OPT_TYPE (MD|DIIS)
Specifies the type optimizer used for the band
This keyword expects a keyword. Default value: DIIS
valid keywords:
  • MD: Molecular dynamics-based optimizer
  • DIIS: Coupled steepest descent / direct inversion in the iterative subspace
OPTIMIZE_END_POINTS
Performs also an optimization of the end points of the band.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
