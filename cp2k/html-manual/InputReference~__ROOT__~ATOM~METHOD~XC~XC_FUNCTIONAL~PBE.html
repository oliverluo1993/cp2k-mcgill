Section PBE

Index of all sections. This section is located at __ROOT__%ATOM%METHOD%XC%XC_FUNCTIONAL%PBE.

Uses the PBE functional. This section can not be repeated.

This section cites following references: [Perdew1996] [Zhang1998] [Perdew2008]

Keyword descriptions

_SECTION_PARAMETERS_
activates the functional
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
PARAMETRIZATION
switches between the different parametrizations of the functional. Note: Beta parameters used have only 5 significant digits, as published. For higher precision and program comparison use section XC/XC_FUNCTIONAL/LIBXC.
This keyword expects a keyword. Default value: ORIG
valid keywords:
  • ORIG: original PBE
  • REVPBE: revised PBE (revPBE)
  • PBESOL: PBE for solids and surfaces (PBEsol)
SCALE_X
scales the exchange part of the functional
This keyword expects a real. Default value: 1.0000000000000000E+00
SCALE_C
scales the correlation part of the functional
This keyword expects a real. Default value: 1.0000000000000000E+00
