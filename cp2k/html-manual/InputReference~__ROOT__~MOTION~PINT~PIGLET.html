Section PIGLET

Index of all sections. This section is located at __ROOT__%MOTION%PINT%PIGLET.

Controls the PI Generalized Langevin Equation thermostat. Needs the exact harmonic integrator. This section can not be repeated.

This section cites following reference: [Ceriotti2012]

Subsections

  • RNG_INIT
  • EXTRA_DOF

Keyword descriptions

NEXTRA_DOF
Number of extra degrees of freedom to ensure markovian dynamics
This keyword expects an integer. Default value: 8
MATRICES_FILE_NAME
Filename containig the raw matrices from https://epfl-cosmo.github.io/gle4md/index.html?page=matrix
This keyword expects a string. Default value: PIGLET.MAT
SMATRIX_INIT
SMATRIX_INIT (CHOLESKY|DIAGONAL)
Select algorithm to initialize piglet S-matrices
This keyword expects a keyword. Default value: CHOLESKY
valid keywords: CHOLESKY, DIAGONAL
