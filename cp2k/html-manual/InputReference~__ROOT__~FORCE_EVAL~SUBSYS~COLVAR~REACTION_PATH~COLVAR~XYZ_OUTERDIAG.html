Section XYZ_OUTERDIAG

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%COLVAR%REACTION_PATH%COLVAR%XYZ_OUTERDIAG.

Section to define the cross term (XA-XA(0))*(XB-XB(0))+(XA-XA(0))*(YB-YB(0)) or part of its components as a collective variable. The final term is given by the product of the components of A with the components of B. This section can not be repeated.

Subsections

  • POINT

Section keywords

  • ATOMS
  • COMPONENT_A
  • COMPONENT_B
  • PBC

Keyword descriptions

ATOMS
ATOMS {integer} {integer}
Specifies the index of the atoms/points A and B.
This keyword expects precisely 2 integers
variants: POINTS
COMPONENT_A
AXIS (XYZ | X | Y | Z | XY| XZ | YZ)
Define the component of the position vector which will be used as a colvar for atom A.
This keyword expects a keyword. Default value: XYZ
valid keywords: XYZ, X, Y, Z, XY, XZ, YZ
COMPONENT_B
AXIS (XYZ | X | Y | Z | XY| XZ | YZ)
Define the component of the position vector which will be used as a colvar for atom B.
This keyword expects a keyword. Default value: XYZ
valid keywords: XYZ, X, Y, Z, XY, XZ, YZ
PBC
PBC
Whether periodic boundary conditions should be applied on the atomic position before computing the colvar or not.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
