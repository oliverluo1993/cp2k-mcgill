Section MATRIX_ITERATE

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%ALMO_SCF%MATRIX_ITERATE.

Controls linear scaling iterative procedure on matrices: inversion, sqrti, etc. High-order Lanczos accelerates convergence provided it can estimate the eigenspectrum correctly. This section can not be repeated.

Subsections

None

  • ORDER_LANCZOS

Keyword descriptions

EPS_TARGET_FACTOR
EPS_TARGET 100.0
Multiplication factor that determines acceptable error in the iterative procedure. Acceptable error = EPS_TARGET_FACTOR * EPS_FILTER
This keyword expects a real. Default value: 1.0000000000000000E+01
EPS_LANCZOS
EPS_LANCZOS 1.0E-4
Threshold for Lanczos eigenvalue estimation.
This keyword expects a real. Default value: 1.0000000000000000E-03
ORDER_LANCZOS
ORDER_LANCZOS 5
Order of the Lanczos estimator. Use 0 to turn off. Do not use 1.
This keyword expects an integer. Default value: 3
MAX_ITER_LANCZOS
MAX_ITER_LANCZOS 64
Maximum number of Lanczos iterations.
This keyword expects an integer. Default value: 128
