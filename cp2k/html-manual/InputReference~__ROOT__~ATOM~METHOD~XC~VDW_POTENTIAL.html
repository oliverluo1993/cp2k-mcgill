Section VDW_POTENTIAL

Index of all sections. This section is located at __ROOT__%ATOM%METHOD%XC%VDW_POTENTIAL.

This section combines all possible additional dispersion corrections to the normal XC functionals. This can be more functionals or simple empirical pair potentials. This section can not be repeated.

This section cites following references: [Grimme2006] [Tran2013]

Keyword descriptions

POTENTIAL_TYPE
POTENTIAL_TYPE (NONE|PAIR_POTENTIAL|NON_LOCAL)
Type of dispersion/vdW functional or potential to use
This keyword expects a keyword. Default value: NONE
valid keywords:
  • NONE: No dispersion/van der Waals functional
  • PAIR_POTENTIAL: Pair potential van der Waals density functional
  • NON_LOCAL: Nonlocal van der Waals density functional
variants: DISPERSION_FUNCTIONAL
