Section ATOM_GROUP

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%PROPERTIES%ET_COUPLING%BECKE_CONSTRAINT_A%ATOM_GROUP.

Define a group of atoms for use in a Becke constraint. Each repetition of this section creates a new constraint. This section can be repeated.

Subsections

None

Section keywords

  • ATOMS
  • COEFF
  • CHARGE_CONSTRAINT

Keyword descriptions

ATOMS
ATOMS {integer} {integer} .. {integer}
Specifies the list of atoms that are included in the constraint group.
This keyword expects a list of integers
COEFF
COEFF 1.0 -1.0
Defines coefficients for the atoms in the list of atoms. Accepts value +/-1.0.
This keyword expects a list of reals, and may repeat
CHARGE_CONSTRAINT
CHARGE_CONSTRAINT
Determines what type of constraint to apply. Default value (true) implies that a total charge density constraint (rho_alpha + rho_beta) is applied. Set to false to enable a magnetization density constraint (rho_alpha - rho_beta)
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
