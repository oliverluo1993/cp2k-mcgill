Section ERI_MME

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%QMMM%IMAGE_CHARGE%ERI_MME.

Parameters for the calculation of periodic electron repulsion integrals (ERI) using the Minimax-Ewald (MME) method. Note: N_MINIMAX is the only parameter to be tuned for accuracy, all other parameters can be left to default. MME method is faster than numerical GPW. This section can not be repeated.

Keyword descriptions

N_MINIMAX
Number of terms in minimax approximation of reciprocal space potential.
This keyword expects an integer. Default value: 53
CUTOFF
User-defined energy cutoff to be used only if DO_CALIBRATE_CUTOFF is set to .FALSE.
This keyword expects a real. Default value: 3.0000000000000000E+02
DO_CALIBRATE_CUTOFF
Whether the energy cutoff shall be calibrated to minimize upper bound error estimate.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
PRINT_CALIB
Print detailed info on calibration.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
DEBUG
debug mode (consistency of summation methods is checked).
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
DEBUG_TOLERANCE
tolerance for rel. numerical error in debug mode.
This keyword expects a real. Default value: 9.9999999999999995E-07
