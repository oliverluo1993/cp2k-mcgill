Section ERI_MME_TEST

Index of all sections. This section is located at __ROOT__%TEST%ERI_MME_TEST.

Parameters for testing the ERI_MME method for electron repulsion integrals. Testing w.r.t. performance and accuracy. This section can not be repeated.

Keyword descriptions

_SECTION_PARAMETERS_
Controls the activation the ERI_MME test.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
TEST_3C
Whether to test 3-center integrals.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
TEST_2C
Whether to test 2-center integrals.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
ABC
ABC 10.000 10.000 10.000
Specify the lengths of the cell vectors A, B, and C.
This keyword expects precisely 3 reals. Default unit: [angstrom].
MIN_NPOS
Minimum number of atomic distances to consider.
This keyword expects an integer. Default value: 8
NREP
Number of repeated calculation of each integral.
This keyword expects an integer. Default value: 1
CHECK_2C_ACCURACY
Whether integrals should be compared to reference values, created on the fly by exact method (G-space sum on grid without minimax approximation). Note: only feasible for not too many integrals and maximum exponent around 10.0.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
LMAX
Maximum total angular momentum quantum number.
This keyword expects an integer. Default value: 6
ZET_MIN
Minimum exponent.
This keyword expects a real. Default value: 1.0000000000000000E-03
ZET_MAX
Maximum exponent.
This keyword expects a real. Default value: 1.0000000000000000E+00
NZET
Number of exponents (logarithmic partition between ZET_MIN and ZET_MAX).
This keyword expects an integer. Default value: 4
NSAMPLE_3C
If NSAMPLE_3C = N, only calculate every Nth 3-center integral.
This keyword expects an integer. Default value: 1
