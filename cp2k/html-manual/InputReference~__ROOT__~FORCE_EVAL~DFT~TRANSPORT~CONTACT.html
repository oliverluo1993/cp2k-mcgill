Section CONTACT

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%TRANSPORT%CONTACT.

Parameters for defining device contacts. This section can be repeated.

Subsections

None

Section keywords

  • BANDWIDTH
  • START
  • N_ATOMS
  • INJECTION_SIGN
  • INJECTING_CONTACT

Keyword descriptions

BANDWIDTH
BANDWIDTH
The number of neighboring unit cells that one unit cell interacts with.
This keyword expects an integer. Default value: 0
START
START
Index of the first atom in the contact unit cell. Set to 0 to define the contact unit cell as the first/last N_ATOMS of the structure (after cutout)
This keyword expects an integer. Default value: 0
N_ATOMS
N_ATOMS
Number of atoms in the contact unit cell.
This keyword expects an integer. Default value: 0
INJECTION_SIGN
INJECTION_SIGN
Contact unit cell interacts with unit cells to the right (positive) or to the left (negative).
This keyword expects a keyword. Default value: POSITIVE
valid keywords:
  • POSITIVE: When the contact unit cell is at the upper left corner of the Hamiltonian.
  • NEGATIVE: When the contact unit cell is at the lower right corner of the Hamiltonian.
INJECTING_CONTACT
whether or not the contact can inject electrons.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
