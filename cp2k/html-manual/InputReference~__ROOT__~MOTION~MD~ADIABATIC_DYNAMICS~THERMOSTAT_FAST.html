Keyword descriptions

TYPE
thermostat NOSE
Specify the thermostat used for the constant temperature ensembles.
This keyword expects a keyword. Default value: NOSE
valid keywords:
  • NOSE: Uses only the Nose-Hoover thermostat.
REGION
REGION (GLOBAL||MOLECULE||MASSIVE||DEFINED||NONE)
Determines the defined region for fast thermostat
This keyword expects a keyword. Default value: GLOBAL
valid keywords: GLOBAL, MOLECULE, MASSIVE, DEFINED, NONE
