Section KG_METHOD

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%KG_METHOD.

Specifies the parameters for a Kim-Gordon-like partitioning into molecular subunits. This section can not be repeated.

This section cites following references: [Iannuzzi2006] [Brelaz1979] [Andermatt2016]

Keyword descriptions

COLORING_METHOD
COLORING_METHOD GREEDY
Which algorithm to use for coloring.
This keyword expects a keyword. Default value: DSATUR
valid keywords:
  • DSATUR: Maximum degree of saturation, relatively accurate
  • GREEDY: Greedy, fast coloring, less accurate
TNADD_METHOD
TNADD_METHOD ATOMIC
Algorithm to use for the calculation of the nonadditive kinetic energy.
This keyword expects a keyword. Default value: EMBEDDING
valid keywords:
  • EMBEDDING: Use full embedding potential (see Iannuzzi et al)
  • ATOMIC: Use sum of atomic model potentials
