Section NMC_MOVES

Index of all sections. This section is located at __ROOT__%MOTION%TMC%NMC_MOVES.

This section specifies the TMC move types, which are performed within the nested Monte Carlo (NMC). thus using the approximate potential. This section can be repeated.

Subsections

  • MOVE_TYPE

Section keywords

  • NR_NMC_STEPS
  • NMC_FILE_NAME
  • PROB
  • INIT_ACC_PROB

Keyword descriptions

NR_NMC_STEPS
NR_NMC_STEPS {INTEGER}
the number of Nested Mont Carlo moves with in one MC move should be huge enough to reach euilibrium state
This keyword expects an integer. Default value: -1
NMC_FILE_NAME
NMC_FILE_NAME {filename}
input file name for the approximate potential for Nested Monte Carlo.
This keyword expects a word
This keyword behaves as a switchxxx_unspecified_xxx . Default value: " "
PROB
PROB {real}
Defines the probability of the NMC move (considering the ration between the selected other moves)the probabilities of the move types in the NMC section defines only the weight within the NMC steps
This keyword expects a real. Default value: 1.0000000000000000E+00
INIT_ACC_PROB
INIT_ACC_PROB {real}
Defines the initial probability of accepting the move.
This keyword expects a real. Default value: 5.0000000000000000E-01
