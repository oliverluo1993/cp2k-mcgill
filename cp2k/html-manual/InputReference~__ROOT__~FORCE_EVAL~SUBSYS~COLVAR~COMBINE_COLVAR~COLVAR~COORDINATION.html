Section COORDINATION

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%COLVAR%COMBINE_COLVAR%COLVAR%COORDINATION.

Section to define the coordination number as a collective variable. This section can not be repeated.

Keyword descriptions

ATOMS_FROM
ATOMS_FROM {integer} {integer} ..
Specify indexes of atoms/points building the coordination variable.
This keyword expects a list of integers, and may repeat
variants: POINTS_FROM
ATOMS_TO
ATOMS_TO {integer} {integer} ..
Specify indexes of atoms/points building the coordination variable.
This keyword expects a list of integers, and may repeat
variants: POINTS_TO
ATOMS_TO_B
ATOMS_TO_B {integer} {integer} ..
For the CV given by the multiplication of two coorination numbers, here specify indexes of the third set of atoms/points.
This keyword expects a list of integers, and may repeat
variants: POINTS_TO_B

R0
R0 {real}
Specify the R0 parameter in the coordination function.
This keyword expects a real. Default unit: [bohr], default value: 3.0000000000000000E+00 bohr
variants: R_0
NN
NN {integer}
Sets the value of the numerator of the exponential factorin the coordination FUNCTION.
This keyword expects an integer. Default value: 6
variants: EXPON_NUMERATOR
ND
ND {integer}

This keyword expects an integer. Default value: 12
variants: EXPON_DENOMINATOR
