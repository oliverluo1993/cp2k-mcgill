Section HBP

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%COLVAR%DISTANCE_FROM_PATH%COLVAR%HBP.

Section to define the hbond wannier centre as a collective variables. This section can not be repeated.

Keyword descriptions

RCUT
RCUT
Parameter used for computing the cutoff radius for searching the wannier centres around an atom
This keyword expects a real. Default unit: [angstrom], default value: 2.8002851809110435E-01 angstrom
SHIFT
SHIFT
Parameter used for shifting each term in the sum
This keyword expects a real. Default value: 5.0000000000000000E-01
NPOINTS
NPOINTS {integer}
The number of points in the path
This keyword expects an integer. Default value: -1
ATOMS
ATOMS {integer} {integer} {integer}
Specifies the indexes of atoms/points defining the bond (Od, H, Oa).
This keyword expects precisely 3 integers, and may repeat
variants: POINTS
