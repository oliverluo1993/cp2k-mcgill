Section PINT

Index of all sections. This section is located at __ROOT__%MOTION%PINT.

Keyword descriptions

P
Specify number beads to use
This keyword expects an integer. Default value: 3
PROC_PER_REPLICA
Specify number of processors to use for each replica
This keyword expects an integer. Default value: 0
NUM_STEPS
Number of steps (if MAX_STEP is not explicitly given the program will perform this number of steps)
This keyword expects an integer. Default value: 3
MAX_STEP
Maximum step number (the program will stop if ITERATION >= MAX_STEP even if NUM_STEPS has not been reached)
This keyword expects an integer. Default value: 10
ITERATION
Specify the iteration number from which it should be counted
This keyword expects an integer. Default value: 0
TEMP
The temperature you want to simulate
This keyword expects a real. Default unit: [K], default value: 3.0000000000000000E+02 K
T_TOL
threshold for the oscillations of the temperature excedeed which the temperature is rescaled. 0 means no rescaling.
This keyword expects a real. Default unit: [K], default value: 0.0000000000000000E+00 K
variants: TEMP_TO
DT
dt 1.0
timestep (might be subdivised in nrespa subtimesteps
This keyword expects a real. Default unit: [fs], default value: 9.9999999999999989E-01 fs
HARM_INT
HARM_INT (NUMERIC|EXACT)
integrator scheme for integrating the harmonic bead springs.
This keyword expects a keyword. Default value: NUMERIC
valid keywords: NUMERIC, EXACT
NRESPA
number of respa steps for the bead for each md step
This keyword expects an integer. Default value: 5
TRANSFORMATION
TRANSFORMATION (NORMAL|STAGE)
Specifies the coordinate transformation to use
This keyword expects a keyword. Default value: NORMAL
valid keywords: NORMAL, STAGE
PROPAGATOR
PROPAGATOR (PIMD|RPMD)
Specifies the real time propagator to use
This keyword expects a keyword. Default value: PIMD
valid keywords: PIMD, RPMD
FIX_CENTROID_POS
Propagate all DOF but the centroid - useful for equilibration of the non-centroid modes (activated only if TRANSFORMATION==NORMAL)
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
