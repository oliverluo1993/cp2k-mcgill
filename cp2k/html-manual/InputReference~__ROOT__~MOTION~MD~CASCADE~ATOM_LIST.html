Index of all sections. This section is located at __ROOT__%MOTION%MD%CASCADE%ATOM_LIST.

Defines a list of atoms for which the initial velocities are modified. This section can not be repeated.

Keyword descriptions

_DEFAULT_KEYWORD_
{{Integer} {Real} {Real} {Real} {Real}}
Defines the list of atoms for which the velocities are modified. Each record consists of the atomic index, the velocity vector, and a weight to define which fraction of the total energy is assigned to the current atom:

Atomic_index vx vy vx Weight
This keyword expects a string, and may repeat
