Section WALLS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%QMMM%WALLS.

Enables Walls for the QM box. This can be used to avoid that QM atoms move out of the QM box. This section can not be repeated.

Subsections

None

Section keywords

  • WALL_SKIN
  • TYPE
  • K

Keyword descriptions

WALL_SKIN
WALL_SKIN
Specify the value of the skin of the Wall in each dimension. The wall's effect is felt when atoms fall within the skin of the Wall.
This keyword expects precisely 3 reals. Default unit: [angstrom], default value: 5.0000000000000000E-01 5.0000000000000000E-01 5.0000000000000000E-01 angstrom
TYPE
TYPE REFLECTIVE
Specifies the type of wall
This keyword expects a keyword. Default value: REFLECTIVE
valid keywords:
  • NONE: No Wall around QM box
  • REFLECTIVE: Reflective Wall around QM box
  • QUADRATIC: Quadratic Wall around QM box
K
K
Specify the value of the the force constant for the quadratic wall
This keyword expects a real. Default unit: [internal_cp2k].
