Section VIBRATIONAL_ANALYSIS

Index of all sections. This section is located at __ROOT__%VIBRATIONAL_ANALYSIS.

Section to setup parameters to perform a Normal Modes, vibrational, or phonon analysis. Vibrations are computed using finite differences, which implies a very tight (e.g. 1E-8) threshold is needed for EPS_SCF to get accurate low frequencies. The analysis assumes a stationary state (minimum or TS), i.e. tight geometry optimization (MAX_FORCE) is needed as well. This section can not be repeated.

Keyword descriptions

DX
Specify the increment to be used to construct the HESSIAN with finite difference method
This keyword expects a real. Default unit: [bohr], default value: 1.0000000000000000E-02 bohr
NPROC_REP
Specify the number of processors to be used per replica environment (for parallel runs). In case of mode selective calculations more than one replica will start a block Davidson algorithm to track more than only one frequency
This keyword expects an integer. Default value: 1
PROC_DIST_TYPE
PROC_DIST_TYPE (INTERLEAVED|BLOCKED)
Specify the topology of the mapping of processors into replicas.
This keyword expects a keyword. Default value: BLOCKED
valid keywords:
  • INTERLEAVED: Interleaved distribution
  • BLOCKED: Blocked distribution
FULLY_PERIODIC
Avoids to clean rotations from the Hessian matrix.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
INTENSITIES
Calculation of the IR-Intensities. Calculation of dipoles has to be specified explicitly
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
THERMOCHEMISTRY
Calculation of the thermochemical data. Valid for molecules in the gas phase.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
TC_TEMPERATURE
tc_temperature 325.0
Temperature for the calculation of the thermochemical data
This keyword expects a real. Default unit: [K], default value: 2.7314999999999998E+02 K
TC_PRESSURE
Pressure for the calculation of the thermochemical data
This keyword expects a real. Default unit: [Pa], default value: 1.0132500000000001E+05 Pa
