Keyword descriptions

ATOMIC_NUMBER
Specify the atomic number
This keyword expects an integer. Default value: 1
ELEMENT
ELEMENT char
Specify the element to be calculated
This keyword expects a word. Default value: H
RUN_TYPE
RUN_TYPE (NONE|ENERGY|BASIS_OPTIMIZATION|PSEUDOPOTENTIAL_OPTIMIZATION)
Type of run that you want to perform [ENERGY,BASIS_OPTIMIZATION,PSEUDOPOTENTIAL_OPTIMIZATION,,...]
This keyword expects a keyword. Default value: ENERGY
valid keywords:
  • NONE: Perform no run
  • ENERGY: Perform energy optimization
  • BASIS_OPTIMIZATION: Perform basis optimization
  • PSEUDOPOTENTIAL_OPTIMIZATION: Perform pseudopotential optimization
COULOMB_INTEGRALS
COULOMB_INTEGRALS (ANALYTIC|SEMI_ANALYTIC|NUMERIC)
Method to calculate Coulomb integrals
This keyword expects a keyword. Default value: NUMERIC
valid keywords:
  • ANALYTIC: Use analytical method
  • SEMI_ANALYTIC: Use semi-analytical method
  • NUMERIC: Use numerical method
EXCHANGE_INTEGRALS
EXCHANGE_INTEGRALS (ANALYTIC|SEMI_ANALYTIC|NUMERIC)
Method to calculate Exchange integrals
This keyword expects a keyword. Default value: NUMERIC
valid keywords:
  • ANALYTIC: Use analytical method
  • SEMI_ANALYTIC: Use semi-analytical method
  • NUMERIC: Use numerical method
CORE
CORE 1s2 ... or CORE [Ne] or CORE none for 0 electron cores
Specifies the core electrons for a pseudopotential
This keyword expects a list of words
ELECTRON_CONFIGURATION
ELECTRON_CONFIGURATION (1) [Ne] 3s2 ...
Specifies the electron configuration. Optional the multiplicity (m) and a core state [XX] can be declared
This keyword expects a list of words, and may repeat
MAX_ANGULAR_MOMENTUM
MAX_ANGULAR_MOMENTUM 3
Specifies the largest angular momentum calculated [0-3]
This keyword expects an integer. Default value: 3
CALCULATE_STATES
CALCULATE_STATES 5 5 5 3
Specifies the number of states calculated per l value
This keyword expects a list of integers. Default value: 0
