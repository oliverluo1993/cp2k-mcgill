Section OPT_RI_BASIS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%KG_METHOD%ENERGY_CORRECTION%XC%WF_CORRELATION%OPT_RI_BASIS.

Parameters influencing the optimization of the RI MP2 basis. Only exponents of non-contracted auxiliary basis can be optimized. An initial RI auxiliary basis has to be specified. This section can not be repeated.

This section cites following reference: [DelBen2013]

Keyword descriptions

DELTA_I_REL
DELTA_I_REL 1.0E-6_dp
Target accuracy in the relative deviation of the amplitudes calculated with and without RI approximation, (more details in Chem.Phys.Lett.294(1998)143).
This keyword expects a real. Default value: 9.9999999999999995E-07
variants: DI_REL
DELTA_RI
DELTA_RI 1.0E-6_dp
Target accuracy in the absolute difference between the RI-MP2 and the exact MP2 energy, DRI=ABS(E_MP2-E_RI-MP2).
This keyword expects a real. Default value: 5.0000000000000004E-06
variants: DRI
EPS_DERIV
EPS_DERIV 1.0E-3_dp
The derivatives of the MP2 energy with respect to the exponents of the basis are calculated numerically. The change in the exponent a_i employed for the numerical evaluation is defined as h_i=EPS_DERIV*a_i.
This keyword expects a real. Default value: 1.0000000000000000E-03
variants: EPS_NUM_DERIV
MAX_ITER
MAX_ITER 100
Specifies the maximum number of steps in the RI basis optimization.
This keyword expects an integer. Default value: 50
variants: MAX_NUM_ITER
NUM_FUNC
NUM_FUNC {number of s func.} {number of p func.} ...
Specifies the number of function, for each angular momentum (s, p, d ...), employed in the automatically generated initial guess. This will be effective only if RI_AUX_BASIS_SET in the KIND section is not specified.
This keyword expects a list of integers. Default value: -1
BASIS_SIZE
BASIS_SIZE (MEDIUM|LARGE|VERY_LARGE)
Specifies the size of the auxiliary basis set automatically generated as initial guess. This will be effective only if RI_AUX_BASIS_SET in the KIND section and NUM_FUNC are not specified.
This keyword expects a keyword. Default value: MEDIUM
valid keywords: MEDIUM, LARGE, VERY_LARGE
