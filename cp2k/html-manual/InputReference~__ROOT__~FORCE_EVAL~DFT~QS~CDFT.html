Section CDFT

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%QS%CDFT.

Parameters needed to set up a CDFT calculation with OT. Constraint is converged in a separate external SCF loop with settings read from the OUTER_SCF section. Supported constraints: Gaussian Hirshfeld (partial) Becke. This section can not be repeated.

This section cites following reference: [Holmberg2017]

Keyword descriptions

TYPE_OF_CONSTRAINT
TYPE_OF_CONSTRAINT (NONE|HIRSHFELD|BECKE)
Specifies the type of constraint used
This keyword expects a keyword. Default value: NONE
valid keywords:
  • NONE: No constraint (disables section)
  • HIRSHFELD: Hirshfeld-type constraint defined by constraint atom centered spherical Gaussians Partial implementation: no forces or coupling. Requires corresponding section.
  • BECKE: Becke constraint. Requires corresponding section in &QS)
REUSE_PRECOND
REUSE_PRECOND yes
Reuse previously built OT preconditioner if SCF converged in PRECOND_FREQ steps or less.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
PRECOND_FREQ
PRECOND_FREQ {int}
See REUSE_PRECOND.
This keyword expects an integer. Default value: 0
MAX_REUSE
MAX_REUSE {int}
Determines how many times a previously built preconditioner can be reused.
This keyword expects an integer. Default value: 0
PURGE_HISTORY
PURGE_HISTORY yes
Purge wavefunction and constraint history to improve SCF convergence during MD.Counts how often the convergence of the first CDFT SCF iteration takes 2 or more outer SCF iterations and purges the history if the counter exceeds PURGE_FREQ and PURGE_OFFSET MD steps have passed since the last purge. The counter is zeroed after each purge.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
PURGE_FREQ
PURGE_FREQ {int}
See PURGE_HISTORY.
This keyword expects an integer. Default value: 1
PURGE_OFFSET
PURGE_OFFSET {int}
See PURGE_HISTORY.
This keyword expects an integer. Default value: 1
COUNTER
COUNTER {int}
A counter to track the total number of energy evaluations. Needed by some optimizers to print information. Useful for restarts.
This keyword expects an integer. Default value: 0
