Section VELOCITY_SOFTENING

Index of all sections. This section is located at __ROOT__%MOTION%MD%VELOCITY_SOFTENING.

A method to initialize the velocities along low-curvature directions in order to favors MD trajectories to cross rapidly over small energy barriers into neighboring basins. In each iteration the forces are calculated at a point y, which is slightly displaced from the current positions x in the direction of the original velocities v. The velocities are then updated with the force component F_t, which is perpendicular to N. N = v / |v|; y = x + delta * N; F_t = F(y) - ⟨ F(y) | N ⟩ * N; v' = v + alpha * F_t. This section can not be repeated.

Keyword descriptions

STEPS
Number of softening iterations performed. Typical values are around 40 steps.
This keyword expects an integer. Default value: 0
DELTA
Displacement used to obtain y.
This keyword expects a real. Default value: 1.0000000000000001E-01
ALPHA
Mixing factor used for updating velocities.
This keyword expects a real. Default value: 1.4999999999999999E-01
