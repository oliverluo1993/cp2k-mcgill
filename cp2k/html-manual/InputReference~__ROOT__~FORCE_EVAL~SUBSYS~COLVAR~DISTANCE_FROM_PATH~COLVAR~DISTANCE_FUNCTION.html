Section DISTANCE_FUNCTION

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%COLVAR%DISTANCE_FROM_PATH%COLVAR%DISTANCE_FUNCTION.

Section to define functions between two distances as collective variables. The function is defined as d1+coeff*d2. This section can not be repeated.

Subsections

  • POINT

Section keywords

  • ATOMS
  • COEFFICIENT
  • PBC

Keyword descriptions

ATOMS
ATOMS {integer} {integer} {integer} {integer}
Specifies the indexes of atoms/points for the two bonds d1=(1-2) d2=(3-4).
This keyword expects precisely 4 integers
variants: POINTS
COEFFICIENT
COEFFICIENT {real}
Specifies the coefficient in the function for the constraint. -1.0 has to be used for distance difference, 1.0 for distance addition
This keyword expects a real
PBC
PBC
Whether periodic boundary conditions should be applied on the atomic position before computing the colvar or not.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
