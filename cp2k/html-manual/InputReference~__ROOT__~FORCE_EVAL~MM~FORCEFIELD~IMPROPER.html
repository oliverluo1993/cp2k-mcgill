Section IMPROPER

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%MM%FORCEFIELD%IMPROPER.

Specifies the improper torsion potential of the MM system. This section can be repeated.

Subsections

None

Keyword descriptions

ATOMS
ATOMS {KIND1} {KIND2} {KIND3} {KIND4}
Defines the atomic kinds involved in the improper tors.
This keyword expects precisely 4 words
KIND
KIND CHARMM
Define the kind of improper torsion potential
This keyword expects a keyword. Default value: CHARMM
valid keywords:
  • CHARMM: Functional Form (CHARMM): K * [ PHI - PHI0 ]**2
  • G87: Functional Form (G87|G96|HARMONIC): 0.5 * K * [ PHI - PHI0 ]**2
  • G96: Functional Form (G87|G96|HARMONIC): 0.5 * K * [ PHI - PHI0 ]**2
  • HARMONIC: Functional Form (G87|G96|HARMONIC): 0.5 * K * [ PHI - PHI0 ]**2
K
K {real}
Defines the force constant of the potential
This keyword expects a real. Default unit: [rad^-2*hartree].
PHI0
PHI0 {real}
Defines the phase of the potential.
This keyword expects a real. Default unit: [rad].
