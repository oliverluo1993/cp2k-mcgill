Section CURVY_STEPS

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%LS_SCF%CURVY_STEPS.

Specifies the parameters of the linear scaling SCF routines. This section can not be repeated.

Keyword descriptions

LINE_SEARCH
LINE Search 3POINT
Line serch type used in the curvy_setp optimization.
This keyword expects a keyword. Default value: 3POINT
valid keywords:
  • 3POINT: Performs a three point line search
  • 3POINT_2D: Only for spin unrestricted calcualtions. Separate step sizes for alpha and beta spin using a fit to a 2D parabolic function
N_BCH_HISTORY
N_BCH_HISTORY 5
Number of stored matrices in the Baker-Campbell-Hausdorff series. Reduces the BCH evaluation during line search but can be memory intense.
This keyword expects an integer. Default value: 7
MIN_HESSIAN_SHIFT
MIN_HESSIAN_SHIFT 0.0
Minimal eigenvalue shift for the Hessian in the Newton iteration. Useful for small band gap systems (0.5-1.0 recommended).
This keyword expects a real. Default value: 0.0000000000000000E+00
FILTER_FACTOR
FILTER_FACTOR 10.0
Allows to set a seperate EPS_FILTER in the newton iterations. The new EPS is EPS_FILTER*FILTER_FACTOR.
This keyword expects a real. Default value: 1.0000000000000000E+00
FILTER_FACTOR_SCALE
FILTER_FACTOR_SCALE 0.5
Allows for dynamic EPS_FILTER. Updates the filter factor every SCF-Newton step by FILTER_FACTOR=FILTER_FACTOR*FILTER_FACTOR_SCALE
This keyword expects a real. Default value: 1.0000000000000000E+00
MIN_FILTER
FILTER_FACTOR 1.0
Lowest EPS_FILTER in dynamic filtering. Given as multiple of EPS_FILTER: EPS_FILTER_MIN=EPS_FILTER*MIN_FILTER
This keyword expects a real. Default value: 1.0000000000000000E+00
