Section CONSTRAINT

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%PROPERTIES%RESP%CONSTRAINT.

specifies a linear constraint on the fitted charges.This can be used to give equal values to equivalent atoms.sum over atom_list c_i * q_i = t. This section can be repeated.

None

Keyword descriptions

TARGET
TARGET 0.0
the target value for the constraint
This keyword expects a real. Default value: 0.0000000000000000E+00
EQUAL_CHARGES
EQUAL_CHARGES
All atoms in ATOM_LIST are constrained to have the same charges. When using this keyword, TARGET and ATOM_COEF do not need to be set and will be ignored. Instead of using this keyword, the constraint section could be repeated.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
ATOM_LIST
ATOM_LIST 3 4
Defines the list of atoms involved in this constraint
This keyword expects a list of integers, and may repeat
ATOM_COEF
ATOM_COEF 1.0 -1.0
Defines the coefficient of the atom in this linear constraint
This keyword expects a list of reals
