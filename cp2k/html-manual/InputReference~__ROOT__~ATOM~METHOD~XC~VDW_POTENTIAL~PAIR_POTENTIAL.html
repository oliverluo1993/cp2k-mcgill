Keyword descriptions

R_CUTOFF
R_CUTOFF 24.0
Range of potential. The cutoff will be 2 times this value
This keyword expects a real. Default unit: [angstrom], default value: 1.0583544171800002E+01 angstrom
TYPE
TYPE (DFTD2|DFTD3|DFTD3(BJ))
Type of potential
This keyword expects a keyword. Default value: DFTD3
valid keywords:
  • DFTD2: Grimme D2 method
  • DFTD3: Grimme D3 method (zero damping)
  • DFTD3(BJ): Grimme D3 method (Becke-Johnson damping)

This keyword cites following references: [Grimme2006] [Grimme2010] [Grimme2011]
PARAMETER_FILE_NAME
PARAMETER_FILE_NAME
Name of the parameter file, may include a path
This keyword expects a string. Default value: DISPERSION_PARAMETERS
REFERENCE_FUNCTIONAL
REFERENCE_FUNCTIONAL
Use parameters for this specific density functional. For available D3 and D3(BJ) parameters see: http://www.thch.uni-bonn.de/tc/downloads/DFT-D3/functionals.html, http://www.thch.uni-bonn.de/tc/downloads/DFT-D3/functionalsbj.html
This keyword expects a word
SCALING
SCALING 0.2
XC Functional dependent scaling parameter, if set to zero CP2K attempts to guess the xc functional that is in use and sets the associated scaling parameter.
This keyword expects a real. Default value: 0.0000000000000000E+00
EXP_PRE
EXP_PRE 20.
Prefactor in exponential damping factor (DFT-D2 potential)
This keyword expects a real. Default value: 2.0000000000000000E+01
EPS_CN
EPS_CN 1.e-6_dp
Cutoff value for coordination number function (DFT-D3 method)
This keyword expects a real. Default value: 9.9999999999999995E-07
D3_SCALING
D3_SCALING 1.0 1.0 1.0
XC Functional dependent scaling parameters (s6,sr6,s8) for the DFT-D3 method, if set to zero CP2K attempts to guess the xc functional from REFERENCE_FUNCTIONAL and sets the associated scaling parameter.
This keyword expects precisely 3 reals. Default value: 0.0000000000000000E+00 0.0000000000000000E+00 0.0000000000000000E+00
D3BJ_SCALING
D3BJ_SCALING 1.0 1.0 1.0 1.0
XC Functional dependent scaling parameters (s6,a1,s8,a2) for the DFT-D3(BJ) method, if set to zero CP2K attempts to guess the xc functional from REFERENCE_FUNCTIONAL and sets the associated scaling parameter.
This keyword expects precisely 4 reals. Default value: 0.0000000000000000E+00 0.0000000000000000E+00 0.0000000000000000E+00 0.0000000000000000E+00
