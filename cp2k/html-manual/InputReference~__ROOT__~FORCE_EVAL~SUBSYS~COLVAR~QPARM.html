Section QPARM

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%COLVAR%QPARM.

Section to define the Q parameter (crystalline order parameter) as a collective variable. This section can not be repeated.

Keyword descriptions

ATOMS_FROM
ATOMS_FROM {integer} {integer} ..
Specify indexes of atoms/points building the coordination variable.
This keyword expects a list of integers, and may repeat
variants: POINTS_FROM
ATOMS_TO
ATOMS_TO {integer} {integer} ..
Specify indexes of atoms/points building the coordination variable.
This keyword expects a list of integers, and may repeat
variants: POINTS_TO
RCUT
RCUT {real}
Specifies the distance cutoff for neighbors. Cutoff function is exactly zero for all neighbors beyond RCUT.
This keyword expects a real. Default unit: [angstrom].
INCLUDE_IMAGES
INCLUDE_IMAGES
Whether to include periodic images of ATOMS_TO into the neighbor list.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
RSTART
RSTART {real}
Specifies the distance cutoff for neighbors. Cutoff function is exactly 1 for all neighbors closer than RSTART.
This keyword expects a real. Default unit: [angstrom].
L
L {integer}
Specifies the L spherical harmonics from Ylm.
This keyword expects an integer
