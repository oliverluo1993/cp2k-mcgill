Keyword descriptions

PARMTYPE
PARMTYPE {OFF,CHM,G87,G96}
Define the kind of torsion potential
This keyword expects a keyword. Default value: OFF
valid keywords:
  • OFF: Provides force field parameters through the input file
  • CHM: Provides force field parameters through an external file with CHARMM format
  • G87: Provides force field parameters through an external file with GROMOS 87 format
  • G96: Provides force field parameters through an external file with GROMOS 96 format
  • AMBER: Provides force field parameters through an external file with AMBER format (from v.8 on)
PARM_FILE_NAME
PARM_FILE_NAME {FILENAME}
Specifies the filename that contains the parameters of the FF.
This keyword expects a string
VDW_SCALE14
VDW_SCALE14 1.0
Scaling factor for the VDW 1-4
This keyword expects a real. Default value: 1.0000000000000000E+00
EI_SCALE14
EI_SCALE14 1.0
Scaling factor for the electrostatics 1-4
This keyword expects a real. Default value: 0.0000000000000000E+00
SHIFT_CUTOFF
SHIFT_CUTOFF
Add a constant energy shift to the real-space non-bonding interactions (both Van der Waals and electrostatic) such that the energy at the cutoff radius is zero. This makes the non-bonding interactions continuous at the cutoff.
This keyword expects a logical. Default value: T
DO_NONBONDED
DO_NONBONDED T
Controls the computation of all the real-space (short-range) nonbonded interactions. This also includes the real-space corrections for excluded or scaled 1-2, 1-3 and 1-4 interactions. When set to F, the neighborlists are not created and all interactions that depend on them are not computed.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
IGNORE_MISSING_CRITICAL_PARAMS
IGNORE_MISSING_BOND_PARAMS T
Do not abort when critical force-field parameters are missing. CP2K will run as if the terms containing the missing parameters are zero.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
MULTIPLE_POTENTIAL
MULTIPLE_POTENTIAL T
Enables the possibility to define NONBONDED and NONBONDED14 as a sum of different kinds of potential. Useful for piecewise defined potentials.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
ZBL_SCATTERING
ZBL_SCATTERING T
A short range repulsive potential is added, to simulate collisions and scattering.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
