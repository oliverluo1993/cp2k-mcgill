Section KPOINT_SET

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%PRINT%BAND_STRUCTURE%KPOINT_SET.

Specifies a k-point line to be calculated. This section can be repeated.

Subsections

None

Keyword descriptions

SPECIAL_POINT
SPECIAL_POINT 0.0 0.0 0.0
Coordinates of a special k-point
This keyword expects precisely 3 reals, and may repeat
NPOINTS
Number of k-points along the line.
This keyword expects an integer. Default value: 0
UNITS
UNITS
Special k-points are defined either in units of reciprocal lattice vectors or in Cartesian coordinates in uints of 2Pi/len. B_VECTOR: in multiples of the reciprocal lattice vectors (b). CART_ANGSTROM: In units of 2*Pi/Angstrom. CART_BOHR: In units of 2*Pi/Bohr.
This keyword expects a word. Default value: B_VECTOR
