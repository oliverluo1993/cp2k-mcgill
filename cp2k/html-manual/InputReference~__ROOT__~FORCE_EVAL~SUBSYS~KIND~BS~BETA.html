Orbital ccupation change per angular momentum quantum number. Applied to spin beta and active only in unrestricted calculations.

L 2
Angular momentum quantum number of the orbitals of beta spin whose occupation is changed. Active only for unrestricted calculations
This keyword expects a list of integers. Default value: -1
N
N 2
Principal quantum number of the orbitals of beta spin whose occupation is changed. Default is the first not occupied. Active only for unrestricted calculations
This keyword expects a list of integers. Default value: 0
