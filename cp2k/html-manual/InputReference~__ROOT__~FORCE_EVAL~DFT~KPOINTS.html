Keyword descriptions

SCHEME
SCHEME {KPMETHOD} {integer} {integer} ..
Kpoint scheme to be used.
Available options: NONE, GAMMA, MONKHORST-PACK, MACDONALD, and GENERAL.
For MONKHORST-PACK and MACDONALD the number of k points in all 3 dimensions has to be supplied along with the keyword. E.g. 
MONKHORST-PACK 12 12 8.
This keyword expects a list of words
This keyword cites following references: [Monkhorst1976] [MacDonald1978]
KPOINT
KPOINT x y z w
Specify kpoint coordinates and weight.
This keyword expects precisely 4 reals, and may repeat
UNITS
UNITS
Special k-points are defined either in units of reciprocal lattice vectors or in Cartesian coordinates in uints of 2Pi/len. B_VECTOR: in multiples of the reciprocal lattice vectors (b). CART_ANGSTROM: In units of 2*Pi/Angstrom. CART_BOHR: In units of 2*Pi/Bohr.
This keyword expects a word. Default value: B_VECTOR
SYMMETRY
SYMMETRY
Use symmetry to reduce the number of kpoints.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
FULL_GRID
FULL_GRID
Use full non-reduced kpoint grid.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
VERBOSE
VERBOSE
Verbose output information.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
EPS_GEO
EPS_GEO
Accuracy in symmetry determination.
This keyword expects a real. Default value: 9.9999999999999995E-07
PARALLEL_GROUP_SIZE
PARALLEL_GROUP_SIZE
Number of processors to be used for a single kpoint. This number must divide the total number of processes. The number of groups must divide the total number of kpoints. Value=-1 (smallest possible number of processes per group, satisfying the constraints). Value=0 (all processes). Value=n (exactly n processes).
This keyword expects an integer. Default value: -1
WAVEFUNCTIONS
WAVEFUNCTIONS REAL
Use real/complex wavefunctions if possible.
This keyword expects a keyword. Default value: COMPLEX
valid keywords:
  • REAL: Use real wavefunctions (if possible by kpoints specified).
  • COMPLEX: Use complex wavefunctions (default).
