Section TDDFPT

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%PROPERTIES%TDDFPT.

Parameters needed to set up the Time-Dependent Density Functional Perturbation Theory. Current implementation works for hybrid functionals. Can be used with Gaussian and Plane Waves (GPW) method only. This section can not be repeated.

This section cites following reference: [Iannuzzi2005]

Keyword descriptions

_SECTION_PARAMETERS_
Controls the activation of the TDDFPT procedure
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
NSTATES
Number of excited states to converge.
This keyword expects an integer. Default value: 1
ADDED_STATES
Number of additional excited states in the active space (-1 means all virtual orbitals).
This keyword expects an integer. Default value: -1
MAX_ITER
Maximal number of iterations to be performed.
This keyword expects an integer. Default value: 50
MAX_KV
Maximal number of Krylov space vectors. Davidson iterations will be restarted upon reaching this limit.
This keyword expects an integer. Default value: 5000
MAX_REORTHO
Maximal number of orthogonalisation iterations. Davidson iterations will be restarted if orthogonality is not reached.
This keyword expects an integer. Default value: 1
NPROC_STATE
Number of MPI processes to be used per excited state. Default is to use all processors (0).
This keyword expects an integer. Default value: 0
CONVERGENCE
Target accuracy for excited state energies.
This keyword expects a real. Default unit: [hartree], default value: 1.0000000000000001E-05 hartree
DEGENERATE_EPS
Energy threshold which controls when excited states are considered to be degenerate.
This keyword expects a real. Default unit: [hartree], default value: 1.0000000000000001E-05 hartree
MIN_AMPLITUDE
The smallest excitation amplitude to print.
This keyword expects a real. Default value: 5.0000000000000003E-02
ORTHOGONAL_EPS
The largest possible overlap between the ground state and orthogonalised excited state wave-functions. Davidson iterations will be restarted when the overlap goes beyond this threshold in order to prevent numerical instability.
This keyword expects a real. Default value: 1.0000000000000000E-04
ADD_DEGENERATE_STATES
Automatically include degenerate excited states into the active space.
This keyword expects a logical. Default value: T
RESTART
Restart the TDDFPT calculation if a restart file exists
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
RKS_TRIPLETS
Compute triplet excited states using spin-unpolarised molecular orbitals.
This keyword expects a logical. Default value: F
WFN_RESTART_FILE_NAME
WFN_RESTART_FILE_NAME
Name of the wave function restart file, may include a path. If no file is specified, the default is to open the file as generated by the wave function restart print key.
This keyword expects a string
variants: RESTART_FILE_NAME
