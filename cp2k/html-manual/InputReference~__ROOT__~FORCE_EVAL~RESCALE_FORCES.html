Section RESCALE_FORCES

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%RESCALE_FORCES.

Section controlling the rescaling of forces. Useful when starting from quite bad geometries with unphysically large forces. This section can not be repeated.

Subsections

None

Section keywords

  • MAX_FORCE

Keyword descriptions

MAX_FORCE
Specify the Maximum Values of the force. If the force of one atom exceed this value it's rescaled to the MAX_FORCE value.
This keyword expects a real. Default unit: [bohr^-1*hartree], default value: 4.2164878378225974E-02 bohr^-1*hartree
