Section PP_BASIS

Index of all sections. This section is located at __ROOT__%ATOM%PP_BASIS.

Keyword descriptions

BASIS_TYPE
BASIS_TYPE (GAUSSIAN|GEOMETRICAL_GTO|CONTRACTED_GTO|SLATER|NUMERICAL)
Basis set type
This keyword expects a keyword. Default value: GAUSSIAN
valid keywords:
  • GAUSSIAN: Gaussian type orbitals
  • GEOMETRICAL_GTO: Geometrical Gaussian type orbitals
  • CONTRACTED_GTO: Contracted Gaussian type orbitals
  • SLATER: Slater-type orbitals
  • NUMERICAL: Numerical basis type
NUM_GTO
NUM_GTO 5 5 5
Number of Gaussian type functions for s, p, d, ...
This keyword expects a list of integers. Default value: -1
NUM_SLATER
NUM_SLATER 5 5 5
Number of Slater type functions for s, p, d, ...
This keyword expects a list of integers. Default value: -1
START_INDEX
START_INDEX 0 2 5 4
Starting index for Geometrical Basis sets
This keyword expects a list of integers. Default value: 0

S_EXPONENTS 1.0 2.0 ...

GEOMETRICAL_FACTOR
GEOMETRICAL_FACTOR real
Geometrical basis: factor C in a*C^k
This keyword expects a real. Default value: 2.6000000000000001E+00
GEO_START_VALUE
GEO_START_VALUE real
Geometrical basis: starting value a in a*C^k
This keyword expects a real. Default value: 1.6000000000000000E-02
BASIS_SET_FILE_NAME
BASIS_SET_FILE_NAME
Name of the basis set file, may include a path
This keyword expects a string. Default value: BASIS_SET
BASIS_SET
BASIS_SET DZVP
The contracted Gaussian basis set
This keyword expects a word. Default value: " "
variants: ORBITAL_BASIS_SET, ORB_BASIS
QUADRATURE
QUADRATURE (GC_SIMPLE|GC_TRANSFORMED|GC_LOG)
Algorithm to construct the atomic radial grids
This keyword expects a keyword. Default value: GC_LOG
valid keywords:
  • GC_SIMPLE: Gauss-Chebyshev quadrature
  • GC_TRANSFORMED: Transformed Gauss-Chebyshev quadrature
  • GC_LOG: Logarithmic transformed Gauss-Chebyshev quadrature
GRID_POINTS
GRID_POINTS integer
Number of radial grid points
This keyword expects an integer. Default value: 400
EPS_EIGENVALUE
EPS_EIGENVALUE real
Cutoff of overlap matrix eigenvalues included into basis
This keyword expects a real. Default value: 9.9999999999999998E-13
