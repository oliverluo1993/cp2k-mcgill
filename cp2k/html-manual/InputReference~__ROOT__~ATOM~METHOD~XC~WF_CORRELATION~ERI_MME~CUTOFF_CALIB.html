Section CUTOFF_CALIB

Index of all sections. This section is located at __ROOT__%ATOM%METHOD%XC%WF_CORRELATION%ERI_MME%CUTOFF_CALIB.

Parameters for the calibration of the energy cutoff by minimizing the errors due to finite cutoff and minimax approximation. Implemented as bisection of error(minimax) - error(cutoff). Not implemented for non-orthorhombic cells. This section can not be repeated.

MAX

This keyword expects a real. Default value: 1.0000000000000000E+04
DELTA
Relative widening of cutoff interval in case starting values are not valid.
This keyword expects a real. Default value: 9.0000000000000002E-01
EPS
Relative cutoff precision required to stop calibration.
This keyword expects a real. Default value: 1.0000000000000000E-02
