Section M-SAMPLING

Index of all sections. This section is located at __ROOT__%MOTION%PINT%HELIUM%M-SAMPLING.

None

Keyword descriptions

DISTRIBUTION-TYPE
DISTRIBUTION-TYPE (SINGLEV|UNIFORM|LINEAR|QUADRATIC|EXPONENTIAL|GAUSSIAN)
Distribution from which the cycle length m is sampled
This keyword expects a keyword. Default value: UNIFORM
valid keywords: SINGLEV, UNIFORM, LINEAR, QUADRATIC, EXPONENTIAL, GAUSSIAN
M-VALUE
Value of m treated in a special way (specific behavior depends on the distribution type chosen)
This keyword expects an integer. Default value: 1
M-RATIO
Probability ratio betw M-VALUE and other cycle lengths
This keyword expects a real. Default value: 1.0000000000000000E+00
