Section BSSE

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%BSSE.

This section is used to set up the BSSE calculation. It also requires that for each atomic kind X a kind X_ghost is present, with the GHOST keyword specified, in addition to the other required fields. This section can not be repeated.
