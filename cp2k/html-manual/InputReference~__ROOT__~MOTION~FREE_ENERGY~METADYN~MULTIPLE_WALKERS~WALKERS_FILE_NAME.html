Section WALKERS_FILE_NAME

Index of all sections. This section is located at __ROOT__%MOTION%FREE_ENERGY%METADYN%MULTIPLE_WALKERS%WALKERS_FILE_NAME.

Specify the basename for the NUMBER_OF_WALKERS files used to communicate between the walkers. Absolute path can be input as well together with the filename. One file will be created for each spawned hill. This section can not be repeated.
