Section MAO

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%DFT%KG_METHOD%ENERGY_CORRECTION%XC%WF_CORRELATION%RI_RPA%IM_TIME%MAO.

Parameters influencing the modified atomic orbitals for cubic RPA. Is only turned on if DO_MAO TRUE in IM_TIME section. This section can not be repeated.

Keyword descriptions

MAO_OCC
MAO_OCC 10 14 6
Specifies the number of MAOs per atom kind for the scaled occupied density matrix. The first MAO number corresponds to the first atom kind. If MAO_OCC is not specified, no MAOs are done for the occupied density matrix.
This keyword expects a list of integers. Default value: -1
MAO_VIRT
MAO_VIRT 10 14 6
Specifies the number of MAOs per atom kind for the scaled virtual density matrix. The first MAO number corresponds to the first atom kind. If MAO_OCC is not specified, no MAOs are done for the virtual density matrix.
This keyword expects a list of integers. Default value: -1
MAX_ITER_OCC
MAX_ITER_OCC 1000
Number of MAO iterations for the MAO basis for the occupied density matrix.
This keyword expects an integer. Default value: 200
MAX_ITER_VIRT
MAX_ITER_VIRT 1000
Number of MAO iterations for the MAO basis for the virtual density matrix.
This keyword expects an integer. Default value: 200
EPS_GRAD_OCC
EPS_GRAD_OCC 1.0E-8
Convergence criterion for creating the MAO basis for the occupied density matrix.
This keyword expects a real. Default value: 9.9999999999999995E-08
EPS_GRAD_VIRT
EPS_GRAD_VIRT 1.0E-8
Convergence criterion for creating the MAO basis for the occupied density matrix.
This keyword expects a real. Default value: 9.9999999999999995E-08
OPTIMIZE_SCALED_OCC_DM
OPT_SC_DM_OCC TRUE
If switched on, the MAOs are optimized to a scaled occupied density matrix.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
variants: OPT_SC_DM_OCC
OPTIMIZE_SCALED_VIRT_DM
OPT_SC_DM_VIRT TRUE
