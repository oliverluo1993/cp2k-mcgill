Keyword descriptions

NUMBER_OF_SHELLS
NUMBER_OF_SHELLS 2
Number of shells taken into account for periodicity. By default, cp2k tries to automatically evaluate this number. This algorithm might be to conservative, resulting in some overhead. You can try to adjust this number in order to make a calculation cheaper.
This keyword expects an integer. Default value: -1
