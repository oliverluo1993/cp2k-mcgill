Section THERMAL_REGION

Index of all sections. This section is located at __ROOT__%MOTION%MD%THERMAL_REGION.

Define regions where different initialization and control of the temperature is used. When MOTION%MD%ENSEMBLE is set to LANGEVIN, this section controls if the atoms defined inside and outside the thermal regions should undergo Langevin MD or NVE Born-Oppenheimer MD. The theory behind Langevin MD using different regions can be found in articles by Kantorovitch et al. listed below. This section can not be repeated.

This section cites following references: [Kantorovich2008] [Kantorovich2008b]

Keyword descriptions

FORCE_RESCALING
Control the rescaling ot the velocities in all the regions, according to the temperature assigned to each reagion, when RESTART_VELOCITY in EXT_RESTART is active.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
DO_LANGEVIN_DEFAULT
do_langevin_default .FALSE.
If ENSEMBLE is set to LANGEVIN, controls whether the atoms NOT defined in the thermal regions to undergo langevin MD or not. If not, then the atoms will undergo NVE Born-Oppenheimer MD.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
