Section BUCKMORSE

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%MM%FORCEFIELD%NONBONDED%BUCKMORSE.

This section specifies the input parameters for Buckingham plus Morse potential type Functional Form: V(r) = F0*(B1+B2)*EXP([A1+A2-r]/[B1+B2])-C/r^6+D*{EXP[-2*beta*(r-R0)]-2*EXP[-beta*(r-R0)]}. This section can be repeated.

This section cites following reference: [Yamada2000]
