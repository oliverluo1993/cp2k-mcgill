Section PAO_POTENTIAL

Index of all sections. This section is located at __ROOT__%FORCE_EVAL%SUBSYS%KIND%PAO_POTENTIAL.

Settings of the PAO potentials, which are atomic kind specific. This section can be repeated.

Subsections

None

Section keywords

  • MAXL
  • BETA
  • WEIGHT
  • MAX_PROJECTOR

Keyword descriptions

MAXL
Maximum angular moment of the potential (must be an even number).
This keyword expects an integer. Default value: 0
BETA
Exponent of the Gaussian potential term.
This keyword expects a real. Default value: 1.0000000000000000E+00
WEIGHT
Weight of Gaussian potential term.
This keyword expects a real. Default value: 1.0000000000000000E+00
MAX_PROJECTOR
Maximum angular moment of the potential's projectors. Used only by the GTH parametrization
This keyword expects an integer. Default value: 2
