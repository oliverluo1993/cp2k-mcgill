Keyword descriptions

_SECTION_PARAMETERS_
Whether or not to actually calculate densities (requires significant amount of memory, depending on the value of NBIN)
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
NBIN
Number of grid points in each direction for density binning
This keyword expects an integer. Default value: 100
MIN_CYCLE_LENGTHS_WDG
..
Density of winding paths not shorter than the given length
This keyword expects a list of integers
MIN_CYCLE_LENGTHS_NON
..
Density of non-winding paths not shorter than the given length
This keyword expects a list of integers
MIN_CYCLE_LENGTHS_ALL
..
Density of all paths not shorter than the given length
This keyword expects a list of integers
ATOM_NUMBER
Atom number density
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
