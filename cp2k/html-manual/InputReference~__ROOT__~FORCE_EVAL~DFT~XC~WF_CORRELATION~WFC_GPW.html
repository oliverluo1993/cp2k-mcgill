Keyword descriptions

EPS_GRID
EPS_GRID 1.0E-9
Determines a threshold for the GPW based integration
This keyword expects a real. Default value: 1.0000000000000000E-08
EPS_FILTER
EPS_FILTER 1.0E-10
Determines a threshold for the DBCSR based multiply (usually 10 times smaller than EPS_GRID).Normally, this EPS_FILTER determines accuracy and timing of cubic-scaling RPA calculation.
This keyword expects a real. Default value: 1.0000000000000001E-09
CUTOFF
CUTOFF 300
The cutoff of the finest grid level in the MP2 gpw integration.
This keyword expects a real. Default value: 3.0000000000000000E+02
REL_CUTOFF
REL_CUTOFF 50
Determines the grid at which a Gaussian is mapped.
This keyword expects a real. Default value: 5.0000000000000000E+01
variants: RELATIVE_CUTOFF
MULTIPOLE_TWO_CENT_INT
MULTIPOLE_TWO_CENT_INT
Use fast algorithm to calculate two-center RI integrals based on multipole expansion
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
PRINT_LEVEL
PRINT_LEVEL HIGH
How much output is written by the individual groups.
This keyword expects a keyword. Default value: SILENT
valid keywords:
  • SILENT: Almost no output
  • LOW: Little output
  • MEDIUM: Quite some output
  • HIGH: Lots of output
  • DEBUG: Everything is written out, useful for debugging purposes only
variants: IOLEVEL
OBARA_SAIKA_OVERLAP
OSO
Use Obara-Saika integrals for two- and three-center integrals.
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
variants: OSO
EPS_PGF_ORB_S
EPS_PGF_ORB_S 1.0E-10
Screening for overlap matrix in RI. Usually, it is best to choose this parameter to be very small since the inversion of overlap matrix might be ill-conditioned.
This keyword expects a real. Default value: 1.0000000000000000E-10
