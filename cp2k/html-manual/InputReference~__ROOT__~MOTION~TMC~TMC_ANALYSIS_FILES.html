Section TMC_ANALYSIS_FILES

Index of all sections. This section is located at __ROOT__%MOTION%TMC%TMC_ANALYSIS_FILES.

Keyword descriptions

RESTART
RESTART {LOGICAL}
Enables/disables the reading and writing of analysis restart files
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
PREFIX_ANA_FILES
ANA_FILES_PREFIX {prefix}
specifies a prefix for all analysis files.
This keyword expects a word. Default value: " "
DENSITY
DENSITY or DENSITY {INTEGER} {INTEGER} {INTEGER}
Mass density in the simulation cell, or if specified in sub cubes
This keyword expects a list of integers. Default value: 1
G_R
G_R or G_R {INTEGER}
Radial Distribution Function for each pair of atomsusing the amount of specified bins within MAX(cell_lenght)/2
This keyword expects an integer
This keyword behaves as a switch -1 . Default value: -1
CLASSICAL_DIPOLE_MOMENTS
CLASSICAL_DIPOLE_MOMENTS or CLASSICAL_DIPOLE_MOMENTS {LOGICAL}
calculates the classical dipole Moment. Following flag specifies if they should be written. Class. Dip. Mom. are also used to unfold the exact dipole moment.
This keyword expects a logical
This keyword behaves as a switch T . Default value: T
DIPOLE_ANALYSIS
DIPOLE_ANALYSIS or DIPOLE_ANALYSIS {type}
Enables analysis of dipole moments, espacially dielectric constant. An additional type can be specified, e.g. analyzing ice structures.using SYM_XYZ also dipoles (-x,y,z) .. .. (-x,-y,z).... (-x,-y-z) are regarded, only use it if this configurations have all the same energy.
This keyword expects a word
This keyword behaves as a switchxxx_unspecified_xxx . Default value: " "
DEVIATION
DEVIATION {LOGICAL}
Calculates the deviation of the position from the last configuration
This keyword expects a logical
This keyword behaves as a switch T . Default value: F
NR_TEMPERATURE
NR_TEMP {INTEGER}
the number of different temperature for parallel tempering
This keyword expects an integer. Default value: 1
TEMPERATURE
TEMPERATURE {REAL} |OR| TEMPERATURE {REAL} {REAL} ...
one temperatue OR for parallel tempering: Tmin Tmax or each temperature T1 T2 T3 ... If every single temperature is specified, do NOT use keyword NR_TEMPERATURE
This keyword expects a list of reals. Default value: 3.3000000000000000E+02
