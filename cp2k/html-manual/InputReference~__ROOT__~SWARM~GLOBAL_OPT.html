Section GLOBAL_OPT

Index of all sections. This section is located at __ROOT__%SWARM%GLOBAL_OPT.

Section to control global geometry optimizations. This section can not be repeated.

Subsections

  • PROGRESS_TRAJECTORY
  • HISTORY
  • MINIMA_HOPPING
  • MINIMA_CRAWLING

Section keywords

  • METHOD
  • E_TARGET
  • MD_BUMPS_MAX
  • BUMP_STEPS_UPWARDS
  • BUMP_STEPS_DOWNWARDS
  • FRAGMENTATION_THRESHOLD

Keyword descriptions

METHOD
Methods to use for optimization.
This keyword expects a keyword. Default value: MINIMA_HOPPING
valid keywords:
  • MINIMA_HOPPING: Runs Minima-Hopping algorithm.
  • MINIMA_CRAWLING: Runs Minima-Crawling algorithm.
E_TARGET
Target Energy, the optimization will quit once a lower potential energy is reached.
This keyword expects a real. Default unit: [hartree], default value: -1.7976931348623157+308 hartree
MD_BUMPS_MAX
Number of bumps in potential energy after which MD runs ends.
This keyword expects an integer. Default value: 3
BUMP_STEPS_UPWARDS
Number of MD steps with potential energy increases required for a bump.
This keyword expects an integer. Default value: 2
BUMP_STEPS_DOWNWARDS
Number of MD steps with potential energy decreases required for a bump.
This keyword expects an integer. Default value: 2
FRAGMENTATION_THRESHOLD
Threshold for atom distance used for detecting fragmentation of clusters.
This keyword expects a real. Default unit: [angstrom], default value: 1.0583544171800001E+00 angstrom
